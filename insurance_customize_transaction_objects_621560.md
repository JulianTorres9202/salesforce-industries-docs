# Customize Transaction Objects

Vlocity provides a Transactions object to support transaction component functionality.

[](https://help.salesforce.com/s?language=en_US)The Transactions object contains a number of custom fields you can use. You can also add your own custom fields based on your business needs. Before you add new custom fields, gather business requirements so you know what types the fields should be, and what user profiles to permit to see these fields.

1.  In Salesforce Classic view, go to Setup > Create > Objects.
2.  Scroll down to the Transactions object, then click its name.
3.  In the Custom Fields & Relationships section, click New.
4.  Follow the Salesforce steps to create a new custom field, then click Save.
