# Manage Picklist Values in the Status Field of the Claim Line Item Object

Deactivate Status picklist values and set the default value of the Status on the Claim Line Item object.

1.  From Setup, in Object Manager, find and select **Claim Line Item**.
2.  Click **Fields & Relationships**.
3.  Click **Status**.
4.  In the **Values** section, deactivate the following values by clicking **Deactivate** and then **OK**:
    
    -   **Reopened**
        
    -   **Pending Approval**
        
    -   **Approved**
        
    -   **Pending Payment**
        
    -   **Payment Approved**
        
    
5.  In the **Values** section, next to the **New** value, click **Edit**.
6.  Select the **Default** checkbox.
7.  Click **Save**.
