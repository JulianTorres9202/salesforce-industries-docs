# Assign Permission Sets to the Analytics Integration User

If your org uses Tableau CRM for Insurance, assign permission sets to the Analytics Integration User.

1.  From **Setup**, enter Users in the Quick Find box, and then select **Users**.
2.  Select the Analytics Integration User.
3.  Go to Permission Set Assignments and then click **Edit Assignments**.
4.  Select the **Vlocity Insurance Analytics Integration** and **Insurance Industries Extension Analytics Integration** permission sets.
5.  Click **Add**, then click **Save**.
