# Post-Install Steps for Property & Casualty OmniStudio

Before exploring the Property & Casualty Application org, you need to complete a few configuration tasks to make sure you don't run into any glitches while you're doing your thing. This section walks you through all the post-deployment steps that you must complete.

-   [Configure the Self-Service Portal](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_the_self-service_portal_omnistudio.htm&language=en_US&type=5 "Any member sites that you've created must be updated and also enabled for standard external profiles so that your policyholders and agents can access them.")
    
    [](https://help.salesforce.com/s?language=en_US)Any member sites that you've created must be updated and also enabled for standard external profiles so that your policyholders and agents can access them.
    
-   [Configure Sharing Rules for Site Users](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_sharing_rules_for_site_users_with_omnistudio.htm&language=en_US&type=5 "Sharing rules give your site users greater access by making automatic exceptions to your org-wide sharing settings.")
    
    [](https://help.salesforce.com/s?language=en_US)Sharing rules give your site users greater access by making automatic exceptions to your org-wide sharing settings.
    
-   [Deactivate and Activate OmniScripts](https://help.salesforce.com/s/articleView?id=ind.insurance_deactivate_and_activate_omniscripts_omnistudio.htm&language=en_US&type=5 "Log in to your org and then deactivate and activate OmniScripts so they can run as intended. For each OmniScript, complete the deactivations and activation tasks.")
    
    [](https://help.salesforce.com/s?language=en_US)Log in to your org and then deactivate and activate OmniScripts so they can run as intended.
    
-   [Configure Renewal Notifications](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_renewal_notifications_omnistudio.htm&language=en_US&type=5 "You can send email notifications for policies that are up for renewal. When sending this email from Salesforce, you can select the organization-wide email address for the email’s From field and notifications are sent to email addresses listed in the member's Account record.")
    
    [](https://help.salesforce.com/s?language=en_US)You can send email notifications for policies that are up for renewal. When sending this email from Salesforce, you can select the organization-wide email address for the email’s **From** field and notifications are sent to email addresses listed in the member's Account record.
    
-   (Optional) [Set Up Google Maps API Key](https://help.salesforce.com/s/articleView?id=ind.insurance_set_up_the_google_map_api_key_omnistudio.htm&language=en_US&type=5 "Use Google Map APIs to automatically populate address fields in the Property and Casualty application.")
    
    [](https://help.salesforce.com/s?language=en_US)Did you know that the Property and Casualty application supports Google Map APIs to automatically populate address fields? By default, this functionality is disabled for all OmniScripts.
    

-   **[Components to Activate](https://help.salesforce.com/s/articleView?id=ind.insurance_components_to_activate.htm&language=en_US&type=5)**  
    Activate the required OmniScripts, FlexCards, flows, and batch jobs for enrollment, termination, and quoting processes.
-   **[Configure the Self-Service Portal](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_the_self-service_portal_omnistudio.htm&language=en_US&type=5)**  
    Any member sites that you've created must be updated and also enabled for standard external profiles so that your policyholders and agents can access them.
-   **[Configure Sharing Rules for Site Users with OmniStudio](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_sharing_rules_for_site_users_with_omnistudio.htm&language=en_US&type=5)**  
    Sharing rules give your site users greater access by making automatic exceptions to your org-wide sharing settings.
-   **[Deactivate and Activate OmniScripts](https://help.salesforce.com/s/articleView?id=ind.insurance_deactivate_and_activate_omniscripts_omnistudio.htm&language=en_US&type=5)**  
    Log in to your org and then deactivate and activate OmniScripts so they can run as intended. For each OmniScript, complete the deactivations and activation tasks.
-   **[Configure Renewal Notifications](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_renewal_notifications_omnistudio.htm&language=en_US&type=5)**  
    You can send email notifications for policies that are up for renewal. When sending this email from Salesforce, you can select the organization-wide email address for the email’s **From** field and notifications are sent to email addresses listed in the member's Account record.
-   **[Set up the Google Map API Key](https://help.salesforce.com/s/articleView?id=ind.insurance_set_up_the_google_map_api_key_omnistudio.htm&language=en_US&type=5)**  
    Use Google Map APIs to automatically populate address fields in the Property and Casualty application.
