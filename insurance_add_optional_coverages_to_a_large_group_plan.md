# Add Optional Coverages to a Large Group Plan

Many plans include optional coverages that you can add and remove from the plan while you're creating a quote.

1.  Go to the plan you want to add optional coverages to and click the arrow next to its name to expand it and view the coverages.
2.  Scroll down. Unselected optional coverages are listed after required coverages and selected optional coverages.
3.  On the optional coverage you want to add, click **Add**.
4.  View the optional coverage details in the side panel.
