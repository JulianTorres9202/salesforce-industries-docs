# Modify Options and Inputs on the Data Source

If the current data source is the correct one, but you need to make changes to specific things like inputs or options, you can do that. Here's how:

1.  Deactivate the FlexCard.
2.  On the **Setup** tab on the FlexCard, go to the **Data Source**.
3.  Make changes to items in the **Input Map**, **Options Map**, **Inputs**, **Options**, or **Headers**.
4.  Click **Save & Fetch**.
5.  Activate the FlexCard.
