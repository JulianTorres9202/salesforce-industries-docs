# Create Product Selling Model Options

To define how you sell products, create product selling model options at the product level.

| User Permissions Needed |
| --- |
| To create product selling model options: | Manage Product Catalog |

1.  From the App Launcher, find and select **Product Catalog Management**.
2.  Select **Products**.
3.  Open the product that you want to add a product selling model option to.
4.  On the Related tab, click **New** on the Product Selling Model Option component.
5.  In Product Selling Model, select an option, and then save your changes.
