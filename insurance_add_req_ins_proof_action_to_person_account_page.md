# Add the Request Insurance Proof Action to the Person Account Page

To give your users quick access to the Request Insurance Proof OmniScript, add the Request Insurance Proof action launcher component to the person account page layout.

1.  In Setup, click **Object Manager**.
2.  In the Quick Find box, enter Account, and then select **Account**.
3.  Click Lightning Record Pages, and select **Account Record Page**.
4.  Click **Edit**.
5.  On the Components tab, add the **Action Launcher** to the record page.
6.  In the properties pane, in Action Launcher Deployment, select the action launcher deployment name that you entered earlier when you created the Request Insurance Proof action.
7.  Save your changes.
