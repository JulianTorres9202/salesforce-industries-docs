# Launch Quick Actions

Configure actions by using flows, OmniScripts, or quick actions. Your customer service representatives can use Action Launcher to quickly find and run service actions, such as uploading documents.

The quick actions available on Insurance Agent Console are:

-   Upload documents
    
-   Update address
    

To learn more about quick actions, see [Action Launcher](https://help.salesforce.com/s/articleView?id=ind.the_action_launcher_component.htm&language=en_US&type=5).
