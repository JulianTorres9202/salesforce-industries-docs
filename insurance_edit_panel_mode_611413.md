# Edit Panel Mode

The Edit Panel Mode is designed for specific use cases where products are highly configurable and can have a large number of coverages and attributes.

Example: Health large group products with up to 100 coverages. With this many coverages, the in-line view is very cumbersome to use. Edit Panel Mode creates a side panel that contains the coverage attributes, making it easier to configure coverages as needed.

The Edit Panel Mode currently does not support adding or configuring insured items and parties.

To use **Edit Panel Mode**, you click a coverage and then work from the side panel that appears.

1.  Click the coverage you want to change attributes for.
    
    A panel opens to the right of the coverage. This panel lists all attribute categories on the coverage. The attribute categories are initially displayed closed, so the system doesn't have to fetch all data at once.
    
2.  Click the attribute category you want to change attributes in.
    
    All the attributes in the category that are associated with the coverage appear.
    
3.  Make changes to attributes.
4.  Click **Save** either at the top or bottom of the right panel.
    
    The changes you made are saved to the quote line item record.
