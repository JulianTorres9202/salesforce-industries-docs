# Search and Filtering Capability

You can filter the split arrangements by type, line of business, or line of coverage from policy, account, or all available split arrangements.

For example, if you select a value for only one of the Type, Line of Business, or Line of Coverage fields, then the split arrangements that are filtered based on the provided value appear. If you select both Type and Line of Business, then the split arrangements that are filtered based on both these fields appear. If you don’t select a value for any of these fields, then all available split arrangements appear.
