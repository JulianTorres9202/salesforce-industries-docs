# Assign Custom Object Permissions for Data Mapper Bundles and Data Mapper Map Items

Assign custom object permissions for the Omnistudio Data Mapper Bundles and Data Mapper Map Items custom objects so that your users can run Data Mapper.

1.  From Setup, enter Users in the Quick Find box, then click **User Management Settings**.
2.  Set the **Enhanced Profile User Interface** option to **Disabled**.
3.  From Setup, enter Users in the Quick Find box, then click **Profiles**.
4.  Find the profile for the users and click **Edit**.
5.  Scroll to the Custom Object Permissions section.
6.  Select the appropriate permissions for the **Vlocity Data Mapper Bundles** and **Vlocity Data Mapper Map Items** custom objects.
    
    Designer users need all permissions. Other users need only the Read permission.
    
7.  Click **Save**.
8.  [](https://help.salesforce.com/s?language=en_US)Repeat steps 3 through 7 for all profiles that need access to Data Mappers.
