# Evaluation of Underwriting Rules

You can evaluate the underwriting rules at runtime for a quote record by using the Insurance Invoke Underwriting Rules connect API.

-   **[Example: Create a Custom Flow for Underwriting Rule Evaluation](https://help.salesforce.com/s/articleView?id=ind.insurance_admin_set_up_execute_underwriting_rule.htm&language=en_US&type=5)**  
    Set up a flow to trigger underwriting rule evaluation. This flow uses an Apex class to call the Underwriting Rules Invoke API directly from the Quote record.
