# Group Benefits Census Management for Quoting in the Salesforce Data Model

The Quote Census Lightning web component for the Salesforce data model shows group members, dependents, and plan selections. Validation rules flag exceptions and team members can add and edit data all in one place. The hierarchical tree view helps brokers and insurance carrier administrators browse through large, detailed data sets more efficiently.

-   **[Get to Know the Census Management Workflow for Group Benefits Quoting](https://help.salesforce.com/s/articleView?id=ind.insurance_get_to_know_the_census_management_workflow_for_group_benefits_quoting.htm&language=en_US&type=5)**  
    The Quote Census Lightning web component gives users a straightforward way to incorporate census data into a quote. After a Salesforce admin configures the component and incorporates it into workflows, insurance carrier administrators and brokers upload census data, identify and correct data exceptions, and turn quotes into contracts.
-   **[Configure Census Management for Group Benefits Quoting](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_census_management_for_group_benefits_quoting.htm&language=en_US&type=5)**  
    Customize the Quote Census Lightning web component for user workflows and create rules that ensure users get the cleanest census data possible.
