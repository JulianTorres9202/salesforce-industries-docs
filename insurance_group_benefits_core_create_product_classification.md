# Set Up Product Classifications in Group Benefits

Set up product classifications to group attributes and provide reusable templates. For example, you can create a GroupSummary classification to model group-level attributes like Total Members and Average Age.

| To create group benefits product model | Product catalog management administrator |

1.  From the App Launcher, find and select **Product Classifications**.
2.  Click **New**.
3.  Enter these field values:
    
    -   Name: GroupSummary
    -   Code: GroupSummary
    -   Status: Active
    
4.  Click **Save**.
