# Configure Commission Statement Line Item Type

Calculate producer commissions by mapping the commission statement line item commission type to the producer split arrangement line item split arrangement type.

| To set up brokerage: | Insurance Brokerage User |

1.  In Setup, find and select **Brokerage**. Then, select **Commissions Management**.
2.  In the Commission Statement Line Type Configuration section, click **Add**.
3.  Specify values for Commission Statement Line Item Commission Type, Producer Split Arrangement Split Type, and Broker Revenue Split Arrangement Type.
    
    To receive the Book of Business revenue credit for a commission type, select a relevant value for Broker Revenue Split Arrangement Type.
    
4.  Save your work.
5.  To update the configuration, click the action menu for the respective record and click **Edit**. Update the values and save your changes.
