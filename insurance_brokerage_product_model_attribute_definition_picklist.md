# Create Attribute Definition with Picklist Data Type

Create a Deductible Waiver attribute definition that’s a picklist with Deductible Waived and Deductible not waived picklist values. Create all attribute definitions with picklist data type. For example, Emergency Room Copay Waived and Retail Copay Multiplier.

1.  From the Product Catalog Management list, select **Attribute Picklists**. Then, click **New**.
2.  Enter the new attribute picklist information and save your changes.
3.  From the Deductible Waiver Picklist record page, click **New** on the Attribute Picklist Values related list.
4.  Enter the attribute picklist value information and save your changes.
    
    The attribute picklist values that you’ve created appear in the Attribute Picklist Values related list.
    
5.  Click the dropdown arrow for Product Catalog Management and select **Attribute Definitions**. Then, click **New**.
6.  Enter the new attribute definition information and save your changes.
