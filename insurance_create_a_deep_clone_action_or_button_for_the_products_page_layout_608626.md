# Create a Deep Clone Action or Button for the Products Page Layout

You can add or create an action or button to the page layout to enable users to clone a product by clicking a button on the Product page.

1.  Go to **Setup > Object Manager > Product > Buttons, Links, and Actions**.
2.  Click **New Button or Link**.
3.  Enter a **Label** and a **Name**.
4.  Select **Detail Page Button** in **Display Type** (what kind of button or link you want) and a **Behavior**.
5.  In **Content Source**, select **Visualforce Page**.
6.  In **Content**, select **InsuranceProductCloner \[vlocity\_ins\_\_InsuranceProductCloner\]**.
7.  Click **Save**.
