# Install the Insurance FSC Unmanaged Package

To start using templates to create an Experience site for your users, install the Insurance FSC unmanaged package.

The highlighted area in this flow shows where you are in the process of setting up your Experience site:

Note

Before you can install the Insurance FSC Unmanaged Package, you need to [enable Digital Experience Sites](https://help.salesforce.com/s/articleView?id=experience.networks_enable.htm&language=en_US&type=5).

1.  Go to the [Insurance Package Information](https://help.salesforce.com/s/articleView?id=ind.insurance_ind_ins_release_summary.htm&language=en_US&type=5 "Insurance includes managed packages, unmanaged packages, and an Installation Assistant that helps you complete installation and upgrade tasks.") and click the link for the Insurance FSC unmanaged package.
2.  On the install window, select **Rename conflicting components in package**.
3.  Select which users will get access to the unmanaged package.
4.  Click **Install**.
