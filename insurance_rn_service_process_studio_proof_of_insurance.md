# Process Proof of Insurance Requests by Using Service Process Studio

Save time and increase customer satisfaction by simplifying the process for requesting insurance proof with the Request Insurance Proof service process. Customers can independently request ‌proof of insurance through self-service portals, or service agents can start the request for them.

**Where:** This change applies to Lightning Experience in Professional, Enterprise, and Unlimited editions where Financial Services Cloud and Financial Services Cloud Insurance is enabled.

**Who:** To use this feature, users need the Industry Service Excellence permission set and the Omnistudio Admin or Omnistudio User permission set. They also need the FSC Insurance permission set license and the Financial Services Cloud Extension, Financial Services Cloud Basic, or Financial Services Cloud Standard permission set license.
