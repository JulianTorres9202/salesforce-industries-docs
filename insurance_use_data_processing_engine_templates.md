# Use Data Processing Engine Templates

The prebuilt data processing engine templates for insurance allow you to create datasets that improve sales and extend customer outreach.

You can customize the Data Processing Engine job templates or create a job template that suits your business needs. You can also schedule a template to run at defined intervals. For more details, refer to the [Data Processing Engine](https://help.salesforce.com/s/articleView?id=sf.concept_data_processing_engine.htm&language=en_US&type=5) help.

-   **[Customer Retention Offers](https://help.salesforce.com/s/articleView?id=ind.insurance_customer_retention_offers.htm&language=en_US&type=5)**  
    Use the Customer Retention Offers Data Processing Engine Template to get a list of customers with active policies with fewer than 10 claims in the last 10 years.
-   **[High-Value Customers](https://help.salesforce.com/s/articleView?id=ind.insurance_high_value_customers.htm&language=en_US&type=5)**  
    Use High-Value Customers Data Processing Engine Template to get a list of accounts with a total premium that's more than $100,000 a year.
-   **[Industry-Related Policies](https://help.salesforce.com/s/articleView?id=ind.insurance_industry_related_policies.htm&language=en_US&type=5)**  
    Use Industry-Related Policies Data Processing Engine Template to get a list of policies related to an industry.
-   **[Lapsed Policies](https://help.salesforce.com/s/articleView?id=ind.insurance_lapsed_policies.htm&language=en_US&type=5)**  
    Use the Lapsed Policies Data Processing Engine template to get a list of lapsed insurance policies.
-   **[Life Event Updates](https://help.salesforce.com/s/articleView?id=ind.insurance_life_event_updates.htm&language=en_US&type=5)**  
    Use the Life Event Updates Data Processing Engine Template to get a list of customer life event updates for the last three months.
-   **[New Sales](https://help.salesforce.com/s/articleView?id=ind.insurance_new_sales.htm&language=en_US&type=5)**  
    Use the New Sales Data Processing Engine template to get a list of new sales that are pending quote to policy conversion.
-   **[Open Claims](https://help.salesforce.com/s/articleView?id=ind.insurance_open_claims.htm&language=en_US&type=5)**  
    Use the Open Claims Data Processing Engine template to get a list of open claims.
-   **[Policy Renewals](https://help.salesforce.com/s/articleView?id=ind.insurance_policy_renewals.htm&language=en_US&type=5)**  
    Use the Policy Renewals Data Processing Engine template to get a list of insurance policies that are up for renewal in the next 30 days.
