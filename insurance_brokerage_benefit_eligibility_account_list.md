# Add an Insurance Benefits Eligibility Definition to an Account.

To associate an Insurance Benefits Eligibility definition with a specific record, add the definition to the account related list.

| User Permissions Needed |

| To use benefit eligibility definitions: | Insurance Policy Management User |

1.  From the App Launcher, find and select **Accounts**, and then select the account for which you want to add an Insurance Benefits Eligibility definition.
2.  From an account record page, on the Insurance Benefits Eligibility Definitions related list, select **New**.
3.  Enter required data.
4.  Save your changes.
