# Create Record Types for Insurance Coexistence

Create unique record types for the Quote, Insurance Policy, and Claim objects to use in Digital Insurance workflows. Use the record types to customize record pages and build custom automations for Digital Insurance.

### Required Editions

The Insurance managed package creates these record types for the Quote and Claim objects.

| Object | Record Types |
| --- | --- |
| Quote | 
-   Quote
-   Small Group Quote
-   Large Group Quote
-   Group Cart

 |
| Claim | P&C Personal Lines |

For Digital Insurance, create unique record types for the Quote, Claim, and Insurance Policy objects. Make sure that the record types differ from those that the managed package creates and any record types that you created for custom workflows with the managed package or Insurance CRM.
