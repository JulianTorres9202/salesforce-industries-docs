# Streamline Requesting Proof Of Insurance by Using Service Process Studio

Use the Service Process Studio to enable the Request Insurance Proof service process to give your agents a simplified process for requesting ‌proof of insurance.

-   **[Required User Permissions to Users](https://help.salesforce.com/s/articleView?id=ind.insurance_required_user_permissions_to_users.htm&language=en_US&type=5)**  
    We recommend that you use permission sets to manage your users' permissions.
-   **[Create a Proof of Insurance Template and Clone the Proof of Insurance Omniscript](https://help.salesforce.com/s/articleView?id=ind.insurance_create_poi_temp_and_clone_the_poi_omniscript.htm&language=en_US&type=5)**  
    Learn how to create a customizable Proof of Insurance template and clone an Omniscript for guided insurance proof requests.
-   **[Activation and Cloning of the Process Insurance Proof Request Flow](https://help.salesforce.com/s/articleView?id=ind.insurance_activation_and_cloning_of_insurance_proof.htm&language=en_US&type=5)**  
    Use the Process Insurance Proof Request Flow Orchestration to show your agents the step-by-step instructions to submit an insurance proof request for a user. You can customize the flow to meet your business needs.
-   **[Case Record Page Management](https://help.salesforce.com/s/articleView?id=ind.insurance_case_record_page_management.htm&language=en_US&type=5)**  
    To show your agents the data attributes that are associated with a service process request for a case, add the case details lightning component to the case record page.
-   **[Create a Request Insurance Proof Service Process](https://help.salesforce.com/s/articleView?id=ind.insurance_create_request_insurance_proof_service_process.htm&language=en_US&type=5)**  
    Create a unique version of the Request Insurance Proof service process and enter the details.
-   **[Add Data Attributes](https://help.salesforce.com/s/articleView?id=ind.insurance_add_data_attributes.htm&language=en_US&type=5)**  
    You can create data attributes to store information from a client. The Request Insurance Proof service process has predefined data attributes. Create more data attributes to suit your business requirements.
-   **[Update the Process Insurance Proof Request Salesforce Flow for Service Catalog Requests](https://help.salesforce.com/s/articleView?id=ind.insurance_update_ins_proof_req_salesforce_flow_for_ser_cat_req.htm&language=en_US&type=5)**  
    To trigger the Process Insurance Proof Request Salesforce flow only for updates in response to Service Catalog Requests, connect the SvcCatalogItemDefinitionID field to your Request Insurance Proof service process. Update the Start element in the cloned Process Insurance Proof Request Salesforce flow and add your Request Insurance Proof service process API name. When a case is created, a Service Catalog Request is created.
-   **[Create a Request Insurance Proof Action](https://help.salesforce.com/s/articleView?id=ind.insurance_create_request_insurance_proof_action.htm&language=en_US&type=5)**  
    To give your users quick access to the Request Insurance Proof Omniscript, create an action launcher deployment that uses the service process template. If you have an existing action launcher deployment, add the Request Insurance Proof Omniscript to it.
-   **[Add the Request Insurance Proof Action to the Person Account Page](https://help.salesforce.com/s/articleView?id=ind.insurance_add_req_ins_proof_action_to_person_account_page.htm&language=en_US&type=5)**  
    To give your users quick access to the Request Insurance Proof OmniScript, add the Request Insurance Proof action launcher component to the person account page layout.
-   **[Default Configuration and Customization for Proof of Insurance Omniscript](https://help.salesforce.com/s/articleView?id=ind.insurance_default_config_and_cust_for_proof_of_insurance_omniscript.htm&language=en_US&type=5)**  
    Understand the default configuration for Proof of Insurance Omniscript and customize it according to your business requirement.
-   **[Required Omniscript Components](https://help.salesforce.com/s/articleView?id=ind.insurance_required_omniscript_components.htm&language=en_US&type=5)**  
    Verify that the required components are in place in Salesforce to support the Request Insurance Proof service process. The Address Update Omniscript calls these Integration Procedures, Omnistudio Data Mappers, FlexCards, and Apex classes.
