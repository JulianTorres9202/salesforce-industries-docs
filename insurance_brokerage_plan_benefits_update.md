# Update Plan Benefits

Update a single benefit record or multiple benefit records.

### Required Editions

| User Permissions Needed |
| --- |
| To update plan benefits: | Insurance Policy Management User |

1.  Go to the Insurance policy coverage record page and select the **Plan Benefits** tab.
2.  To update a single benefit, click the action menu of the benefit record that you want to update and select **Edit**.
3.  Update the details and save your changes.
4.  To update more than one benefit record, select all the records that you want to update and click **Edit**.
5.  In the Edit Benefits window, update the details.
6.  Save your changes.
