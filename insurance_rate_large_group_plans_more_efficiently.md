# Rate Large Group Plans More Efficiently

To help your users rate more plans at a time, use the updated version of the Generate Census Rating flow and the Finish Processing Census Rating flow.

**Where:** This change applies to Lightning Experience in Professional, Enterprise, and Unlimited editions with the Insurance Industries managed package and the Insurance Industries Extension managed package.

**How:** If you used the Generate Census Rating and Finish Processing Census Rating flows with the Insurance Winter ‘24 package, deactivate the existing flow versions and configure the new Spring ‘24 versions. After you install the Insurance Spring ‘24 package, the Winter ‘24 versions of the large group quoting census rating flows don’t work.
