## Automatic Creation of Opportunities

An Opportunity is automatically generated when you create an Endorsement (Mid-Term Adjustment) quote. The new Opportunity generated has the same name as the Endorsement quote and the Opportunity type is set to **Endorsement**.

-   **[Create Quote Versions](https://help.salesforce.com/s/articleView?id=ind.insurance_create_quote_versions_612919.htm&language=en_US&type=5)**  
    To generate a new version of a quote, create a version from the Quote record.
-   **[Compare Quote Versions](https://help.salesforce.com/s/articleView?id=ind.insurance_compare_quote_versions_612946.htm&language=en_US&type=5)**  
    You can compare versions of a quote on a Quote record.
