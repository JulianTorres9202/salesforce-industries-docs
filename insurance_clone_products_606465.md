# Clone Products

If you need to create a bunch of similar product models that contain many of the same child specs and attributes and rules, it's much quicker to clone and modify a product than it is to model each one from scratch.

1.  Clone a product.
    1.  Go to the Products page and click the product you want to clone.
    2.  On the Product Detail page, click the clone button created in the previous task.
    3.  Click **Continue** on the message that appears.
    4.  On the new (cloned) product, make any needed changes to any part of the product.
2.  Change the product code assigned to the cloned product.
    1.  Go to the Details tab on the Product Details page of the cloned product.
    2.  Click **Edit** at the top right side of the page.
    3.  By default, the cloned product's product code = <original product code>CLONED. We recommend you change the Product Code to a unique set of characters that doesn't include the word CLONED.
    4.  Click **Save**.
3.  Customize the new product model. On a cloned product model, you can make any changes you want to any part of the product.
