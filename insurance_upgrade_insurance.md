# Upgrade Insurance

If you already use Insurance and you're ready to upgrade to a recent release, you're in the right place. Learn how to prepare and complete your upgrade so you can start using all the great features now available.

[](https://help.salesforce.com/s?language=en_US)

## Upgrade Workflow

1.  [Complete Prerequisite Tasks](https://help.salesforce.com/s/articleView?id=ind.insurance_vlocity_insurance_and_vlocity_health_pre_install_and_pre_upgrade_tasks.htm&language=en_US&type=5#insurance_vlocity_insurance_and_vlocity_health_pre_install_and_pre_upgrade_tasks "Insurance uses some Salesforce features that aren't enabled by default in a typical org. To ensure Insurance works correctly with Salesforce, complete certain tasks and verify certain settings before you install or upgrade Insurance packages.")
2.  [Plan Your Upgrade](https://help.salesforce.com/s/articleView?id=ind.insurance_plan_your_upgrade_asof242.htm&language=en_US&type=5 "Before you upgrade, verify a few details about your current org and figure out exactly which release to upgrade to.")
3.  [Install packages](https://help.salesforce.com/s/articleView?id=ind.insurance_package_installation_flows_upgrades_asof242.htm&language=en_US&type=5 "Your release includes new managed packages. If you plan to use the Insurance Industries Extension package, you have two managed packages to install. Otherwise, you have just one package to install."):
    
    -   [Install Managed Packages for Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_install_managed_packages_asof242.htm&language=en_US&type=5 "Managed packages are the first packages you install in your Insurance install or upgrade process.")
    
    Find the most recent package links in the [Insurance Release Summary](https://help.salesforce.com/s/articleView?id=ind.insurance_release_summary.htm&language=en_US&type=5#insurance_release_summary "Get release summaries for Insurance minor releases delivered each Winter, Spring, and Summer.").
    
    There isn't a new Unmanaged Package or Vlocity Installation Assistant Package for the Summer ’24 release. To upgrade your org to Summer ’24, the only thing to install is the Managed Package (or Packages).
    
4.  [Complete Installation Assistant Tasks Manually](https://help.salesforce.com/s/articleView?id=ind.insurance_complete_installation_assistant_tasks_manually.htm&language=en_US&type=5 "If you're upgrading from several versions back, you can simplify your upgrade by focusing on changes between releases and by completing certain tasks only one time.")
    
    [](https://help.salesforce.com/s?language=en_US)There are specific steps for an Insurance org, and additional steps for orgs with the Insurance Industries Extension package.
    
    Note If you're upgrading from Spring ’23 or later release to Winter ’25, and you're positive you completed the Installation Assistant tasks for Spring ’23 or later, you can skip the Installation Assistant tasks for Winter ’25. There are no changes to instructions for Winter ’25, so as long as you completed all the steps for the Spring ’23 or later release, your Winter ’25 org is ready to go. If you're not sure whether you completed Installation Assistant tasks for Spring ’23 or later release, it doesn't hurt to complete them again.
    
5.  [Complete Post-Upgrade Tasks](https://help.salesforce.com/s/articleView?id=ind.insurance_post_upgrade_tasks_asof242.htm&language=en_US&type=5 "To complete your upgrade, perform required post-upgrade tasks.")

-   **[Plan Your Upgrade to Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_plan_your_upgrade_asof242.htm&language=en_US&type=5)**  
    Before you upgrade, verify a few details about your current org and figure out exactly which release to upgrade to.
-   **[Package Installation Flows for Insurance Upgrades](https://help.salesforce.com/s/articleView?id=ind.insurance_package_installation_flows_upgrades_asof242.htm&language=en_US&type=5)**  
    Your release includes new managed packages. If you plan to use the Insurance Industries Extension package, you have two managed packages to install. Otherwise, you have just one package to install.
-   **[Complete Installation Assistant Tasks Manually](https://help.salesforce.com/s/articleView?id=ind.insurance_complete_installation_assistant_tasks_manually.htm&language=en_US&type=5)**  
    If you're upgrading from several versions back, you can simplify your upgrade by focusing on changes between releases and by completing certain tasks only one time.
-   **[Post-Upgrade Tasks for Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_post_upgrade_tasks_asof242.htm&language=en_US&type=5)**  
    To complete your upgrade, perform required post-upgrade tasks.
