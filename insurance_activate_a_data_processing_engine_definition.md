# Activate a Data Processing Engine Definition

After you’ve cloned the Data Processing Engine definitions, you must activate them.

1.  From Setup, enter Data Processing Engine in the Quick Find box, and then select **Data Processing Engine** under Workflow Services.
2.  Click the job name that you customized.
3.  Click **Activate** on the builder header.
4.  You can run an active definition with flows or from within the Data Processing Engine builder. If you run a definition from within the builder, define input variables specific to that definition.
