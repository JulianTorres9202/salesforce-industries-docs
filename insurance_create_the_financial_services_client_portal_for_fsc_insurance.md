# Set Up the Insurance Policyholder Portal

Set up the Insurance Policyholder Portal using the Financial Services template in Experience Cloud and configure the self-service capabilities.

To set up the insurance policyholder portal:

-   [Create a Customer Community User Profile](https://help.salesforce.com/s/articleView?id=ind.insurance_create_a_customer_community_user_profile.htm&language=en_US&type=5 "To control access to Insurance features, create a profile for portal users with the appropriate permissions by cloning a community user profile.")
    
-   [Create Portal Users](https://help.salesforce.com/s/articleView?id=ind.insurance_create_portal_users.htm&language=en_US&type=5 "Create portal users for the Insurance Policyholder Portal.")
    
-   [Assign a Permission Set to Portal Users](https://help.salesforce.com/s/articleView?id=ind.insurance_assign_a_permission_set_to_portal_users.htm&language=en_US&type=5 "Create and assign a permission set for portal users to grant the license and permissions they need to access the Insurance Policyholder Portal.")
    
-   [Grant Portal Users Required User and Object Permissions](https://help.salesforce.com/s/articleView?id=ind.insurance_grant_portal_users_required_user_and_object_permissions.htm&language=en_US&type=5 "Grant your portal users the needed user and object permissions to create service requests, schedule appointments, and browse knowledge articles in the Insurance Policyholder Portal.")
    
-   [Create an Insurance Policyholder Portal](https://help.salesforce.com/s/articleView?id=ind.insurance_create_a_self_service_fsc_client_portal_for_fsc_insurance.htm&language=en_US&type=5 "Work with a Financial Services Experience Builder template to create a self-service portal for your customers. Perform initial setup and branding, configure components, and then deploy a working portal to your sandbox.")
    
-   [Configure Self-Service Capabilities](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_self_service_capabilities.htm&language=en_US&type=5 "Use the Experience Builder components and Workspaces in the Financial Services template to set up the self-service capabilities.")
    

-   **[Create a Customer Community User Profile](https://help.salesforce.com/s/articleView?id=ind.insurance_create_a_customer_community_user_profile.htm&language=en_US&type=5)**  
    To control access to Insurance features, create a profile for portal users with the appropriate permissions by cloning a community user profile.
-   **[Create Portal Users](https://help.salesforce.com/s/articleView?id=ind.insurance_create_portal_users.htm&language=en_US&type=5)**  
    Create portal users for the Insurance Policyholder Portal.
-   **[Assign a Permission Set to Portal Users](https://help.salesforce.com/s/articleView?id=ind.insurance_assign_a_permission_set_to_portal_users.htm&language=en_US&type=5)**  
    Create and assign a permission set for portal users to grant the license and permissions they need to access the Insurance Policyholder Portal.
-   **[Grant Portal Users Required User and Object Permissions](https://help.salesforce.com/s/articleView?id=ind.insurance_grant_portal_users_required_user_and_object_permissions.htm&language=en_US&type=5)**  
    Grant your portal users the needed user and object permissions to create service requests, schedule appointments, and browse knowledge articles in the Insurance Policyholder Portal.
-   **[Create an Insurance Policyholder Portal](https://help.salesforce.com/s/articleView?id=ind.insurance_create_a_self_service_fsc_client_portal_for_fsc_insurance.htm&language=en_US&type=5)**  
    Work with a Financial Services Experience Builder template to create a self-service portal for your customers. Perform initial setup and branding, configure components, and then deploy a working portal to your sandbox.
-   **[Experience Cloud Site Setup Checklist for Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_experience_cloud_site_setup_checklist__for_fsc_insurance.htm&language=en_US&type=5)**  
    Use this checklist as a guide to perform Experience Cloud site setup tasks, to brand the home page and to update component properties.
-   **[Configure Self-Service Capabilities](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_self_service_capabilities.htm&language=en_US&type=5)**  
    Use the Experience Builder components and Workspaces in the Financial Services template to set up the self-service capabilities.
