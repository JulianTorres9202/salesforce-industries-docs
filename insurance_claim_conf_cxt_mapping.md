1.  From Setup, in the Quick Find box, enter **Context Service**, and then select **Configure Context Mapping**.
2.  Click **New Context Mapping**.
3.  From the Context Definition list, select a context definition.
    
    The Context Mapping field is automatically filled with the context definition's default mapping.
    
4.  From the Context Mapping field, select a different mapping, if needed.
5.  Click **Active**.
6.  Save your changes.
