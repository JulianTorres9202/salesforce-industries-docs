# insQuoteInsuredItems Lightning Web Component

The insQuoteInsuredItems component, called **Vlocity Insurance Quote Insured Items** in the Lightning App Builder, shows users all the insured items in a quote.

[](https://help.salesforce.com/s?language=en_US)

[](https://help.salesforce.com/s?language=en_US)This component works for desktop and is mobile responsive.

[](https://help.salesforce.com/s?language=en_US)

Here's what it looks like in the user interface:

The **insQuoteInsuredItems** LWC displays:

1.  An icon for the insured item type
    
2.  The name of the insured item
    
3.  Attributes of the insured item
    
4.  Option to edit the insured item
    
5.  A filter for insured items
    

[](https://help.salesforce.com/s?language=en_US)

## What Users Can Do on this LWC

-   See all insured items for a quote
    
-   See details of insured items
    
-   Edit insured item details
    
-   Filter insured items based on product type
    

[](https://help.salesforce.com/s?language=en_US)

## What this LWC Does

-   Updates the quote line item when a user changes an attribute value
    
-   Displays messages to re-rate the Quote when an insured item attribute has changed
    

[](https://help.salesforce.com/s?language=en_US)

## Services Called by this LWC

**insQuoteInsuredItems** does not call any services.

[](https://help.salesforce.com/s?language=en_US)

## Other LWCs Used by this Component

**insQuoteInsuredItems** uses these LWCs:

-   **insUtility**
    
-   **insProductInstance**
    
-   **insRecordEditor**
