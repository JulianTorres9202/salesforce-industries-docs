# Specify Rating Procedure on Root Product Specs

To get the correct pricing for each insurance product, assign a rating procedure to each root product spec.

1.  Go to the root Product spec page, click the **Details** tab, and then click **Edit**.
    
    If you don't see an **Edit** button, find it in the Actions menu.
    
2.  Enter a **Rating Procedure Type**.
    
    -   Expression Set
        
        Expression sets are the calculation engine of business rules.
        
    -   Calculation Procedure
        
        Calculation procedures are part of the Industries packages.
        
    -   Integration Procedure
        
    
3.  Enter a **Rating Procedure Name**.
    
    -   For expression sets, enter the expression set name.
        
    -   For calculation procedures, enter the calculation procedure name.
        
    -   For Integration Procedures, enter the Type\_Subtype of the Integration Procedure.
        
    
4.  Click **Save**.
