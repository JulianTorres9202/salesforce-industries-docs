# Create a Customer Community User Profile

To control access to Insurance features, create a profile for portal users with the appropriate permissions by cloning a community user profile.

1.  From Setup, in the Quick Find box, enter Profiles, and then select **Profiles**.
2.  Next to the user profile that maps to your customer community license, click **Clone**. If your community license is Customer Community Plus, select either Customer Community Plus User or Customer Community Plus Login User as the user profile.
3.  Enter a name for the cloned profile, for example Insurance Customer Community Plus Login User, and then save your changes.
