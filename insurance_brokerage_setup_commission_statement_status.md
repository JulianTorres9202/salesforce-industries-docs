# Map Commission Statement Status

Map the system status of the commission statement to the customer-facing status of the commission statement.

### Required Editions

| User Permissions Needed |
| --- |
| To set up brokerage: | Insurance Brokerage User |

1.  In Setup, find and select **Brokerage**. Then, select **Commissions Management**.
2.  In the Commission Statement Status Mapping section, from the dropdown menu for Customer-Facing Status, select a status to map it with the relevant system status value.
    
    The Customer-Facing Status values are inherited from the Commission Statement object. You can modify these values by updating the Status field in the Commission Statement object.
    
3.  Save your changes.
