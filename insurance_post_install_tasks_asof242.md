# Post-Installation Tasks for Insurance

You need to perform a few additional steps to complete your installation.

-   **[Assign Custom Object Permissions for Data Mapper Bundles and Data Mapper Map Items](https://help.salesforce.com/s/articleView?id=ind.insurance_assign_custom_object_permissions_dm_bundles_and_dm_map_items.htm&language=en_US&type=5)**  
    Assign custom object permissions for the Omnistudio Data Mapper Bundles and Data Mapper Map Items custom objects so that your users can run Data Mapper.
-   **[Activate Record Pages for the Unmanaged Package](https://help.salesforce.com/s/articleView?id=ind.insurance_activate_record_pages_for_the_unmanaged_package.htm&language=en_US&type=5)**  
    Many of the pre-configured record pages are installed automatically in the unmanaged package, however, you still need to clone and activate them.
-   **[Assign Permission Set Licenses for Insurance Industries Extension Objects](https://help.salesforce.com/s/articleView?id=ind.insurance_assign_psl_for_insurance_industries_extension_objects.htm&language=en_US&type=5)**  
    To give your users access to Policy Administration and Claim Management Insurance Industries Extension objects, you must assign the **Policy Administration** _and_ the **Claim Management** Permission Set Licenses.
-   **[Required Permissions for the Partner Community User Profile](https://help.salesforce.com/s/articleView?id=ind.insurance_required_permissions_for_the_partner_community_user_profile.htm&language=en_US&type=5)**  
    You can add a number of permissions to the Partner Community User profile to grant Partner Community Users access to important information.
-   **[Automatic Vlocity State Transition Component Refresh](https://help.salesforce.com/s/articleView?id=ind.insurance_automatic_vlocity_state_transition_component_refresh_2.htm&language=en_US&type=5)**  
    Use a PushTopic to automatically refresh the Vlocity State Transition component when the Status field of a record updates.
-   **[Field-Level Security Permission Set for Insurance Industries Extension Objects and Fields in Installations](https://help.salesforce.com/s/articleView?id=ind.insurance_field_level_security_perm_set_iie_installs_asof242.htm&language=en_US&type=5)**  
    Provide your users with access to Insurance Industries Extension objects and fields by creating a permission set.
-   **[Complete Post-Install Steps for Contract Lifecycle Management for Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_complete_post_install_steps_for_clm_insurance_asof242.htm&language=en_US&type=5)**  
    Contract Lifecycle Management gets installed when you install the Insurance managed package. To configure Contract Lifecycle Management, complete additional tasks.
