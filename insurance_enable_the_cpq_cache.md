# Enable the CPQ Cache

To help your census rating flows run efficiently, enable the CPQ Cache after you enable the CPQPartition platform cache.

1.  From Setup, in the Quick Find box, enter Custom Settings, and then select **Custom Settings**.
2.  Next to CPQ Configuration Setup, click **Manage**.
3.  Click **New**.
4.  Enter these details, and then click **Save**.
    
    | Field | Value |
    | --- | --- |
    | Name | CacheEnabled |
    | Setup Value | true |
