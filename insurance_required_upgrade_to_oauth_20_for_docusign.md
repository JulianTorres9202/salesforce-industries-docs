# Required Upgrade to OAuth 2.0 for DocuSign Integration in Insurance

For Vlocity Document Generation, you must upgrade to OAuth 2.0 to manage DocuSign.

[](https://help.salesforce.com/s?language=en_US)

## OAuth 2.0 Details

DocuSign has deprecated OAuth 1.0 authentication for the more secure OAuth 2.0. Industries Managed Package License users on Winter '23, released on January 10, 2023 (240.11) or later, must configure authentication provider credentials and add named credentials for DocuSign OAuth 2.0 by March 2024 to allow Document Generation to continue using DocuSign in Insurance, Communications, Media, and Energy and Public Sector managed packages.

Where: This change applies to Lightning Experience in Enterprise, Performance, and Unlimited editions.
