# Add Expression Set to the Rating Procedure Type Picklist for the Product Record Type

Add Expression Set to the Rating Procedure Type picklist for the Product record type in the Product object.

1.  From **Setup**, click the **Object Manager** tab.
2.  Click **Product**.
3.  Click **Record Types**.
4.  Click **Product**.
5.  In the **Picklists Available for Editing** section, click **Edit** next to **Rating Procedure Type**.
6.  Add **Expression Set** to **Selected Values**.
7.  Click **Save**.
