# Group Classes for Large Group Quotes

Group classes describe a group of plan subscribers who'll all receive similar or the same benefits. You can add them to plans on large group quotes.

The modals you use to add group classes use a field set shipped out of the box. You can add to this field set if needed. The field set is Group Class Details, on the Group Class object.

You can add or remove a group class for a product on a large group quote.

1.  On the Large Group Quote tab of the quote Ul, go to the product you want to add a class to.
2.  Click **\+ Group Class**.
3.  On the Group Classes modal, select one or more group classes.
4.  Click **Add**.
    
    The group class(es) you added appear as buttons under the product name.
    
5.  To remove a group class from a product, click the button of the group class to remove. Then select **Remove From** ProductName.
    
    The group class is removed from the product.
