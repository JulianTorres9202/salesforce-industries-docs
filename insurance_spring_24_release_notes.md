# Insurance Spring '24 Release Notes

Use flows to verify callers and producers, rate large group quotes, and enroll census members in plans. Delete census members in bulk, enable customers to self-enroll, and identify the correct pricing procedure to use during the policy lifecycle with the help of new and enhanced services. Help agents navigate claim records with the Insurance Agent Console.

-   **[Automatic Upgrades for Managed Packages](https://help.salesforce.com/s/articleView?id=ind.insurance_automatic_upgrades_for_managed_packages.htm&language=en_US&type=5)**  
    Starting in Spring '24, we've added the Insurance managed packages to the automatic upgrade process. This process aligns with the Salesforce seasonal release cycle, where new features are delivered three times a year (Summer, Winter, and Spring). With automatic upgrades, you also get early access to upcoming releases in your sandbox preview environment.
-   **[Use Data Processing Engine Definitions for Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_use_data_processing_engine_definitions_for_insurance.htm&language=en_US&type=5)**  
    To help sales users define their actionable lists for extended outreach, use the predefined insurance data processing engine definitions.
-   **[Insurance Agent Console Enhancements](https://help.salesforce.com/s/articleView?id=ind.insurance_insurance_agent_console_enhancements.htm&language=en_US&type=5)**  
    View claim requests and show alerts for claim record changes that require action with the Insurance Agent Console.
-   **[Large Group Quoting and Enrollment Enhancements](https://help.salesforce.com/s/articleView?id=ind.insurance_large_group_quoting_and_enrollment_enhancements.htm&language=en_US&type=5)**  
    Use enhanced flows to rate large group quotes and enroll members in plans. Delete census members in bulk and enable customers to self-enroll by using new and enhanced services.
-   **[Policy Administration Enhancements](https://help.salesforce.com/s/articleView?id=ind.insurance_policy_administration_enhancements.htm&language=en_US&type=5)**  
    Perform out-of-sequence transactions for multi-term policies, and generate transaction breakdowns and payment schedules. Identify the appropriate pricing procedure to use during policy updates by using the `ratingDate` option in updated services.
-   **[Support Identity Verification By Using Flows](https://help.salesforce.com/s/articleView?id=ind.insurance_support_identity_verification_by_using_flows.htm&language=en_US&type=5)**  
    Quickly verify callers and producers by using prebuilt flows.
-   **[Services for Insurance Spring '24](https://help.salesforce.com/s/articleView?id=ind.insurance_services_for_insurance_spring_24.htm&language=en_US&type=5)**  
    Explore new services and updates to existing services in Spring '24.
-   **[Explore New Features with the Insurance Application Suite](https://help.salesforce.com/s/articleView?id=ind.insurance_explore_new_features_with_the_insurance_application_suite.htm&language=en_US&type=5)**  
    Get hands-on experience with Insurance features from the Spring ’24 release. Learn about large group quoting, enrollment, and mid-term adjustments, and commercial quoting. Find out how to use claims financial authority, and a claims detail page for your policyholders.
