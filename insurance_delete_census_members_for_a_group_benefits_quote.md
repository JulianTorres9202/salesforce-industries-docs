# Delete Census Members for a Group Benefits Quote

Use the Insurance Quote Census Lightning web component in the Salesforce data model to delete members on a Quote record page. You can delete a single census member record or multiple member records using the Delete Members option. The hierarchical tree view helps brokers and insurance carrier administrators browse through large, detailed data sets more efficiently.

Note

Select at least one member record to enable the **Delete Members** icon.

**Where:** This change applies to Lightning Experience in Professional, Enterprise, and Unlimited editions with the Insurance Industries managed package.

**Who:** Financial Services Cloud users must have the Insurance Contract and Enroll add-on license. Health Cloud users must have the Benefits Sales & Administration - Group add-on license.
