# Install Insurance

Install Insurance for the first time.

Important Do not install the Insurance managed package if the OmniStudio managed package is already installed. Having both packages installed results in duplicate components.

If your org already has the OmniStudio package installed and you want to install Insurance, contact Salesforce support for help.

[](https://help.salesforce.com/s?language=en_US)

## Installation Workflow

There are many things you need to do to fully install Insurance so that you can use all of the great features it has to offer. Here's everything you need to do to get up and running:

1.  [Complete Pre-Install Tasks](https://help.salesforce.com/s/articleView?id=ind.insurance_vlocity_insurance_and_vlocity_health_pre_install_and_pre_upgrade_tasks.htm&language=en_US&type=5#insurance_vlocity_insurance_and_vlocity_health_pre_install_and_pre_upgrade_tasks "Insurance uses some Salesforce features that aren't enabled by default in a typical org. To ensure Insurance works correctly with Salesforce, complete certain tasks and verify certain settings before you install or upgrade Insurance packages.")
2.  [Install Packages](https://help.salesforce.com/s/articleView?id=ind.insurance_package_installation_flows.htm&language=en_US&type=5 "Insurance uses managed and unmanaged packages that you install in a certain order. The number of packages you install depends on which features you plan to use."):
    
    1.  [Managed Packages](https://help.salesforce.com/s/articleView?id=ind.insurance_install_managed_packages_asof242.htm&language=en_US&type=5 "Managed packages are the first packages you install in your Insurance install or upgrade process.")
    2.  [Unmanaged Packages](https://help.salesforce.com/s/articleView?id=ind.insurance_install_unmanaged_packages_asof242.htm&language=en_US&type=5 "The Insurance unmanaged package contains custom record pages. The Insurance FSC unmanaged package contains Digital Experience Site templates.")
    3.  [Vlocity Installation Assistant](https://help.salesforce.com/s/articleView?id=ind.insurance_install_the_installation_assistant_asof242.htm&language=en_US&type=5#insurance_install_the_installation_assistant_asof242 "To make it easier to configure Insurance after installing or upgrading, install the unmanaged package that contains the Installation Assistant.")
    
    Note The Insurance managed package includes OmniStudio features like OmniScripts, FlexCards, Data Mappers, and Integration Procedures. Installing the package automatically configures your org's [Prerequisite Tasks for Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_vlocity_insurance_and_vlocity_health_pre_install_and_pre_upgrade_tasks.htm&language=en_US&type=5#insurance_vlocity_insurance_and_vlocity_health_pre_install_and_pre_upgrade_tasks "Insurance uses some Salesforce features that aren't enabled by default in a typical org. To ensure Insurance works correctly with Salesforce, complete certain tasks and verify certain settings before you install or upgrade Insurance packages.") based on your OmniStudio license.
    
    If you install the Insurance managed package, you don't need to also install the standalone OmniStudio package. The Insurance managed package already includes OmniStudio features.
    
    Find the most recent package links in the [Insurance Release Summary](https://help.salesforce.com/s/articleView?id=ind.insurance_release_summary.htm&language=en_US&type=5#insurance_release_summary "Get release summaries for Insurance minor releases delivered each Winter, Spring, and Summer.").
    
3.  [Complete Vlocity Installation Assistant Steps](https://help.salesforce.com/s/articleView?id=ind.insurance_complete_vlocity_installation_assistant_steps.htm&language=en_US&type=5 "To complete the installation and the configuration of Insurance do all the tasks marked New Installation Only and all the tasks not specially marked.")
4.  [Complete Post-Installation Tasks](https://help.salesforce.com/s/articleView?id=ind.insurance_post_install_tasks_asof242.htm&language=en_US&type=5 "You need to perform a few additional steps to complete your installation.")

-   **[Package Installation Flows](https://help.salesforce.com/s/articleView?id=ind.insurance_package_installation_flows.htm&language=en_US&type=5)**  
    Insurance uses managed and unmanaged packages that you install in a certain order. The number of packages you install depends on which features you plan to use.
-   **[Complete Vlocity Installation Assistant Steps](https://help.salesforce.com/s/articleView?id=ind.insurance_complete_vlocity_installation_assistant_steps.htm&language=en_US&type=5)**  
    To complete the installation and the configuration of Insurance do all the tasks marked New Installation Only and all the tasks not specially marked.
-   **[Post-Installation Tasks for Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_post_install_tasks_asof242.htm&language=en_US&type=5)**  
    You need to perform a few additional steps to complete your installation.
