# Add Fields to Object Field Sets for Insurance Release Spring '21 Through Winter '25

Add fields to field sets on several objects used with Insurance.

1.  From **Setup**, click the **Object Manager** tab.
2.  Click the object.
3.  Click **Field Sets**.
4.  Click the Field Set Field Label.
5.  Drag the field(s) from the top panel down into the **In This Field Set** area.
6.  Click **Save**.
