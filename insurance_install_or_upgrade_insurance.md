# Installation and Upgrades

Learn how to install or upgrade Insurance in your org. Your tasks include pre-requisite steps, package installation, Installation Assistant steps, and post-install or post-upgrade tasks.

For a complete list of Insurance packages available, see [Insurance Package Information](https://help.salesforce.com/s/articleView?id=ind.insurance_ind_ins_release_summary.htm&language=en_US&type=5 "Insurance includes managed packages, unmanaged packages, and an Installation Assistant that helps you complete installation and upgrade tasks.").

## Installation and Upgrade Instructions

1.  Complete [Prerequisite Tasks for Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_vlocity_insurance_and_vlocity_health_pre_install_and_pre_upgrade_tasks.htm&language=en_US&type=5#insurance_vlocity_insurance_and_vlocity_health_pre_install_and_pre_upgrade_tasks "Insurance uses some Salesforce features that aren't enabled by default in a typical org. To ensure Insurance works correctly with Salesforce, complete certain tasks and verify certain settings before you install or upgrade Insurance packages.").
2.  If you're installing for the first time, see [Install Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_install_insurance.htm&language=en_US&type=5 "Install Insurance for the first time."). Otherwise see [Upgrade Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_upgrade_insurance.htm&language=en_US&type=5 "If you already use Insurance and you're ready to upgrade to a recent release, you're in the right place. Learn how to prepare and complete your upgrade so you can start using all the great features now available.").
