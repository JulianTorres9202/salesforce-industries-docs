# Generate a Summary

Obtain a summary on the Account or Claim record page by using the Generate functionality. The summary includes insurance policy details, active coverages, or claim specifics. This comprehensive overview enhances agent efficiency by reducing time spent in searching for information.

1.  Navigate to the record's detail page.
2.  From the dropdown on Einstein Summary in the record page, select the appropriate prompt template.
3.  Click **Generate**.
    
    Einstein Summary generates and shows the summary.
    
4.  To copy the summary, click **Copy**.
