# Harmonized Interaction Launcher and Console

The processes in the interaction launcher now write to the native Financial Services Cloud Insurance Policy object instead of the Asset object.

Where: This feature is available in the Insurance Application Suite under Voluntary Benefits Application. The harmonized workflows are available to anyone with a Salesforce FSC license.

[](https://help.salesforce.com/s?language=en_US)Why: We're working to create one unified data model for Vlocity Insurance built on Salesforce FSC. While you don't have to move to this data model, going forward our new features will be built on it.

How: This component is now harmonized with the Financial Services Cloud Insurance Policy object. Your insurance policy data is stored on standard Insurance Policy objects using standard fields. The Harmonized Interaction Launcher and Console is built using different OmniScripts, Data Mappers and Integration Procedures.
