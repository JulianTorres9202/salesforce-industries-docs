# Set the Custom Attributes Process Option on Products

Expose the Custom Attributes Process on the Products page (if it's not already available on the page). This option tells the Quote UI that it should use the OmniScript rather than the out-of-the-box form to add and modify insured items to a quote.

1.  From **Setup**, click the **Object Manager** tab.
2.  Select **Product**.
3.  Click **Page Layouts**.
4.  Click the name of the Product page layout you're using.
5.  On the list of **Fields**, find **Custom Attributes Process** and drag it down onto the **Product Detail** area.
6.  Click **Save**.
    
    After **Custom Attributes Process** is exposed on the Products page, you can select it in root product specs to have those products use your OmniScript in the Quote UI.
