# Track the Results of Underwriting Rule Evaluations

View the history of underwriting rule execution on the Stage Transition Underwriting Evaluations page.

Before reviewing the rule evaluation results, you must enable the Stage Transition Underwriting Evaluations tab.

1.  Enable the Stage Transition Underwriting Evaluations tab.[](https://help.salesforce.com/s?language=en_US)
    1.  From Setup, in the Quick Find box, enter users and then select **Profiles**.
    2.  Select the **Edit** link to the left of the custom profile that you will update.
    3.  Scroll down to Tab Settings and select **Default On** against Stage Transition Underwriting Evaluations.
2.  From the App Launcher, find and select **Stage Transition Underwriting Evaluations**.
3.  Review rule evaluation details including the reference object, evaluation result, date and time, and state transition.
