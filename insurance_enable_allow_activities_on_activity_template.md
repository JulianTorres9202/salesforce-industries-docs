# Enable Allow Activities on Activity Template

Before you upgrade Insurance, enable the Allow Activities setting on the Activity Template object.

1.  From Setup, click the **Object Manager** tab.
2.  Click Activity Template.
3.  Click Edit.
4.  Scroll down to the **Optional Features** section and verify that **Allow Activities** is selected.
5.  If **Allow Activities** is not selected:
    1.  Select **Allow Activities**.
    2.  Click **Save**.
