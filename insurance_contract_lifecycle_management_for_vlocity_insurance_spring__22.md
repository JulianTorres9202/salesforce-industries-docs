# Contract Lifecycle Management For Vlocity Insurance Spring '22

This overview describes the key features and enhancements for Contract Lifecycle Management for Vlocity Insurance in this release.

-   **[Enable Server-Side Document Generation](https://help.salesforce.com/s/articleView?id=ind.insurance_enable_server_side_document_generation.htm&language=en_US&type=5)**  
    System administrators can now enable the server-side document generation feature for new customers who have Document Generation licenses auto-provisioned. Document Generation licenses are auto-provisioned only for new customers who have purchased the new Salesforce SKUs post 17 May 2022, and installed the Salesforce Industries Spring '22 package. Existing customers upgrading to Spring '22, must continue to contact the Salesforce Support to enable the server-side document generation feature.
-   **[Provide CLM and DocGen Users More Access with Permission Set Licenses for Vlocity Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_provide_clm_and_docgen_users_more_access_with_permission_set_licenses_for_vlocity_insurance_and_vlocity_health.htm&language=en_US&type=5)**  
    Starting Spring ’22, new customers can use permission set licenses to incrementally extend the functionality of user licenses. CLM, and DocGen offer permission set licenses for Vlocity Insurance. With permission set licenses, you can assign more permissions to users than what their user license supports.
-   **[Sample Permission Sets for Vlocity Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_sample_permission_sets_for_vlocity_insurance_and_vlocity_health.htm&language=en_US&type=5)**  
    Use the sample permission sets to give your users access to only those features that are essential for performing their tasks, such as read-only access or create and edit record access. Each permission set grants a different level of access to a different set of objects and fields. You can customize the permission sets. Salesforce recommends sample permission sets for Vlocity Insurance to easily grant the right access to your users.
