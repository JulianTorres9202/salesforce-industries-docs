# 2023 Insurance Release Notes

See these release notes for the Insurance releases in 2023.

-   **[Insurance Summer '23 Release Notes](https://help.salesforce.com/s/articleView?id=ind.insurance_summer_23_release_notes.htm&language=en_US&type=5)**  
    Show the right product details to exactly the right users with profile-based and user-specific rules. Simplify workflows with the enhanced Policy Term Standing and Group Benefits Enrollment Lightning web components. Streamline expense processing for claims adjusters with coverage-specific attributes. Reduce errors by indicating required attributes on Claim Involved Injuries and Claim Damage Property record types.
-   **[Insurance Spring '23 Release Notes](https://help.salesforce.com/s/articleView?id=ind.insurance_spring_23_release_notes.htm&language=en_US&type=5)**  
    Expedite financial authorization requests with a workflow that routes them directly to supervisors. Define user-specific attribute values with profile-based attribute rules. Backdate policy changes with out-of-sequence endorsement quoting. Delete census data for group benefits directly on the quote record page with the delete functions on the Insurance Census Lightning web component.
-   **[Insurance Winter '23 Release Notes](https://help.salesforce.com/s/articleView?id=ind.insurance_winter_23_release_notes-.htm&language=en_US&type=5)**  
    Boost quote turnaround and simplify insurance policy administration with enhanced rules and simplified rating sets for coverages and insured items. New Lightning web components streamline Group Benefits quoting and enrollment processes. New services and objects support claim recovery and workers' compensation management.
