# Agent Permission Sets

Give your agents access to just what they need to complete their jobs with Agent Permission Sets.

An agent is a partner user who creates, reviews, and revises quotes, and issues and modifies policies for their customers. But an agent doesn't need access to everything in your org. Agents really only need access to a few objects, and to only certain fields on those objects. We've created permission sets that you can assign to agents to give them access only to the information they need to help their customers succeed—and nothing else.

[](https://help.salesforce.com/s?language=en_US)

## What Can an Agent Do?

What an agent can do depends on which permission set you assign to them. We created eight permission sets with agents in mind so that you can quickly and easily grant just the right access to your agents. Some permission sets provide read-only access, while others grant the ability to create and edit records.

After you have assigned agent permission sets that grants create and edit access to an agent, they can:

-   Create and edit opportunities associated with their parent account.
    
-   Create quotes for their customers.
    
-   Edit quotes and create quote versions to help their customer get exactly the coverage they are looking for.
    
-   Issue policies from quotes and modify existing policies.
    

[](https://help.salesforce.com/s?language=en_US)

## Types of Agent Permission Sets

We've created agent permission sets for you. Each permission set grants a different level of access to a different set of objects and fields.

-   Insurance Create/Update Policy Permission Set
    
-   Insurance Create/Update Quote Permission Set
    
-   Insurance View Policy Permission Set
    
-   Insurance View Quote Permission Set
    
-   Insurance Create/Update Census Data Permission Set
    
-   Insurance Create/Update Enrollment Permission Set
    
-   Insurance View Census Data Permission Set
    
-   Insurance View Enrollment Permission Set
    
-   Insurance View Group Census Quote Permission Set
    
-   Insurance View Group Census Data Permission Set
    
-   Insurance Create/Update Group Census Quote Permission Set
    
-   Insurance Create/Update Group Census Data Permission Set
    

These permission sets just cover the basics. You can modify any of these permission sets to meet the needs of your agents. Read [User Permissions and Access](https://developer.salesforce.com/docs/atlas.en-us.securityimplguide.meta/securityimplguide/permissions_about_users_access.htm) to learn how.
