# Enable Insurance Industries Extension

The Insurance Industries Extension package is an add-on application that requires Financial Services Cloud (FSC) licenses from Salesforce. Before you can use it, you need to enable it.

1.  From Setup, use Quick Find to find **Financial Services**, then select **Insurance Settings**.
2.  Enable **Access Master List of Coverage Types**.
3.  Enable **Use Many-to-Many Relationships**.
