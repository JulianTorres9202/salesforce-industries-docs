# Configure Preferences for Insurance Policy Lifecycle

Configure the fields and context definition for the policy lifecycle process.

See [context definition](https://help.salesforce.com/s/articleView?id=ind.context_service_context_definitions.htm&language=en_US&type=5) and [context](https://help.salesforce.com/s/articleView?id=ind.context_service_map_context_definitions.htm&language=en_US&type=5) mapping for more information on how to create and map context defintiion.

1.  In Setup, find and select **Brokerage**. Select **Policy Management**, and then select **Policy Lifecycle**.
2.  In the Insurance Policy Lifecycle Configuration section, click the action menu for the policy action you want to update and select **Edit**.
3.  Update the fields to show on the insurance policy page and select the context definition for each action.
    
    Ensure that the context definition is active when using the custom context definition and also add the context definition developer name.
    
4.  Save your changes.
