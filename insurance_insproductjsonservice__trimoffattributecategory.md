# InsProductJSONService:trimOffAttributeCategory

Use this service to get a product JSON that does not include attribute categories, but does include attribute selected values, attribute rules, and attribute value rules.

[](https://help.salesforce.com/s?language=en_US)

Class: `InsProductJSONService`

Method: `trimOffAttributeCategory`

[](https://help.salesforce.com/s?language=en_US)This service is intended to be used to create an output JSON that batch processes (that don't include any UI interaction) can use.

[](https://help.salesforce.com/s?language=en_US)This service can also be used when you need to map attribute values to additional fields in the Quote Line Item object.

[](https://help.salesforce.com/s?language=en_US)

## How It Works

1.  Uses the `inputKey` to identify a standard product JSON with one root product as the input JSON.
    
2.  Removes the attribute categories from the product JSON.
    
3.  If the selectedValueAsObject option is set to true, returns the product JSON with attributeSelectedValues as an object map.
    
    If selectedValueAsObject is set to false, returns the attributeSelectedValues as a string.
    

[](https://help.salesforce.com/s?language=en_US)

## Remote Options

| 
Option

 | 

Description

 |
| --- | --- |
| 

`inputKey`

 | 

inputKeyValue

[](https://help.salesforce.com/s?language=en_US)The key the service uses to find the area of the input JSON to trim attribute categories from.

 |
| 

`selectedValueAsObject`

 | 

`true` or `false`

When set to `true`, the service returns attributeSelectedValues as a mapped object instead of a string. This lets you convert the JSONResult to easily map attribute values to additional fields in the Quote Line Item object.

When set to `false` (or if this option is not present), the service returns attributeSelectedValues as a string.

 |
