# Create Claim Product Classifications

A product classification is a template that you use to create products with similar characteristics. Product classifications are the primary tool for assigning attributes to products.

1.  From the Product Catalog Management app's home page, click **Product Classifications**.
2.  On the Product Classifications list view page, click **New**.
3.  In the New Product Classification window, enter these field values:
    1.  Enter a product classification name and code. For example, Auto Claim Root Class and AutoClaimRootClass.
    2.  Select the status as **Active**.
    3.  Click **Save & New**.
4.  Repeat Step 3 to create the other product classifications. The details are:
    
    | product classification name | code | status |
    | --- | --- | --- |
    | Involved Injury Class | InvolvedInjuryClass | Active |
    | Damaged Property Class | DamagedPropertyClass | Active |
