# Invocable Actions Added for Insurance Winter '24

Use the new invocable actions for Insurance.

-   **Generate user inputs and fact ratings**
    
    Use the [Generate User Inputs](https://help.salesforce.com/s/articleView?id=ind.insurance_generate_user_inputs_invocable_action.htm&language=en_US&type=5 "Generates user inputs and fact ratings.") action
    
-   **Rate insurance products as part of the Group Benefits Rating business process**
    
    Use the [Rate Products](https://help.salesforce.com/s/articleView?id=ind.insurance_rate_products_invocable_action.htm&language=en_US&type=5 "Rates insurance products as part of the Group Benefits Rating business process.") action.
    
-   **Create contact records for group census members**
    
    Use the [Create Contacts](https://help.salesforce.com/s/articleView?id=ind.insurance_create_contacts_invocable_action.htm&language=en_US&type=5 "Creates contact records for group census members.") action.
    
-   **Create person account records of a specified type for group census members**
    
    Use the [Create Accounts](https://help.salesforce.com/s/articleView?id=ind.insurance_create_accounts_invocable_action.htm&language=en_US&type=5 "Create person account records of the specified type for group census members.") action.
    
-   **Create user records with the IsPortalUser property enabled for group census members**
    
    Use the [Create Portal Users](https://help.salesforce.com/s/articleView?id=ind.insurance_create_portal_users_invocable_action.htm&language=en_US&type=5 "Creates user records with the IsPortalUser property enabled for group census members.") action.
    
-   **Create insurance policies for specified primary members and their dependents by enrolling them into subscribed plans**
    
    Use the [Enroll Members](https://help.salesforce.com/s/articleView?id=ind.insurance_enroll_members_invocable_action.htm&language=en_US&type=5 "Creates insurance policies for specified primary members and their dependents by enrolling them into subscribed plans.") action.
