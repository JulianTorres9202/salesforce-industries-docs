# Configure the Generate Census Rating Flow

Use this flow to compile available products for group census members and generate rating processes.

1.  From Setup, in the Quick Find box, enter Flows, and then select **Flows**.
2.  Open the Generate Census Rating flow.
3.  Click **Save As**. Add a flow label, API name, and description.
4.  Save and activate the flow.
    
    Note Make sure that you have only one clone of this flow template activated.
