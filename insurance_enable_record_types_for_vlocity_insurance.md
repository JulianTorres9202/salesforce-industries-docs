# Enable Record Types for Insurance

Enabling record types for profiles gives users with that profile access to the record type. Use these steps to enable the Insurance record types listed in the table below.

1.  From **Setup**, in the **Quick Find** box, enter Profiles.
2.  Click on the name of a profile, then scroll down to the **Record Type Settings** section.
3.  Locate the record type in either the **Standard Record Type Settings** or **Custom Record Type Settings** sub-sections, then click **Edit**.
4.  Select all of the record types that the profile needs to access, then click **Add**.
5.  Select a **Default** record type.
6.  Click **Save**.
    
    Repeat these steps for all of these record types:
