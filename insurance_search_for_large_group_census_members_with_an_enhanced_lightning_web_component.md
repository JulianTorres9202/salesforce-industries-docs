# Quickly Find Large Group Census Members During Enrollment and Quoting

Quickly find census member details during large group enrollment and quoting processes by using the Search box on Enrollment Census and Quote Census Lightning web components. To update any related census member records, retrieve and show members of the same family in a dynamic tree within search results to enhance your user experience.

**Where**: This change applies to Lightning Experience in Professional, Enterprise, and Unlimited editions with the Insurance Industries managed package and the Insurance Industries Extension managed package.

**How**: Use the Search box on the Enrollment Census and Quote Census Lightning web components.
