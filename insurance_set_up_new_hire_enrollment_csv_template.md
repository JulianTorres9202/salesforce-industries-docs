# Set Up New Hire Enrollment CSV Template

Use the New Hire Enrollment CSV template to enter the enrollment details for the new hire enrollment process.

1.  From Setup, in the Quick Find box enter Static Resources, and then select **Static Resources**.
2.  Click **New**.
3.  In the **Name** field, enter a name.
4.  In the **File** field, click **Browse** and upload a new hire enrollment CSV file template.
5.  Click **Save**.
6.  Specify the static resource name in the **New Hire CSV Template** field on the account page.
7.  If you want to use the Person Account flow for enrollment, select **Person Account Enabled**.
8.  Click **Save**.
