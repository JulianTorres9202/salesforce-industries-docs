# Add Data Attributes

You can create data attributes to store information from a client. The Request Insurance Proof service process has predefined data attributes. Create more data attributes to suit your business requirements.

1.  In Setup, in the Quick Find box, enter Service Process Studio, and then select **Service Process Studio**.
2.  Edit a service process or create one.
3.  Select **Data Attributes**, and then click **New Data Attribute**.
4.  Enter an attribute name and an API name.
5.  Select an attribute type.
6.  Select a data type.
7.  If needed, select a parent attribute.
8.  If needed, enter a default value and description.
9.  To make this attribute mandatory for record creation, select **Required**.
10.  Click **Add**.
11.  Save your changes.
