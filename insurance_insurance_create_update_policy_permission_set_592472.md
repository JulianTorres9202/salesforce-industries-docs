# Insurance Create/Update Policy Permission Set

Give your agents access to just what they need to create and update insurance policies.

[](https://help.salesforce.com/s?language=en_US)

## Access Provided by the Insurance Create/Update Policy Permission Set

This permission set is delivered in the Insurance Industries Extension managed package and provides Read access to the following objects:

[](https://help.salesforce.com/s?language=en_US)

-   Accounts
    
-   Attribute Assignments
    
-   Compiled Attribute Overrides
    
-   Contacts
    
-   Opportunities
    
-   Price Books
    
-   Producer Commissions
    
-   Producers
    
-   Product Child Items
    
-   Production Codes
    
-   Product Requirements
    
-   Products
    
-   Production Taxes and Fees
    
-   Quotes
    
-   Taxes and Fees
    
-   Vlocity Actions
    
-   Vlocity Attribute Categories
    
-   Vlocity Attributes
    
-   Vlocity State Models
    
-   Vlocity State Model Versions
    
-   Vlocity States
    
-   Vlocity State Transitions
    
-   Vlocity State Transition Rules
    

This permission set provides Read and Create access to the following object:

-   Customer Properties
    

This permission set provides Read, Edit, and Create access to the following objects:

-   Insurance Policies
    
-   Insurance Policy Assets
    
-   Insurance Policy Coverages
    
-   Insurance Policy Member Assets
    
-   Insurance Policy Participants
    
-   Insurance Policy Surcharges
    
-   Insurance Policy Terms
    
-   Insurance Policy Transactions
    

The Insurance Create/Update Policy permission set also provide Read, Edit, Create, and Delete access to these objects:

-   Insurance Policy Payment Schedule
    
-   Insurance Policy Revenue Schedule
    

Additionally, this permission set grants access to certain fields on some of the objects listed above.
