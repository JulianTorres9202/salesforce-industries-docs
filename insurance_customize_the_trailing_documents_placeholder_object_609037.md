# Customize the Trailing Documents Placeholder Object

Configure the Trailing Document Placeholder object to suit your business needs. The object has several custom fields and you can add more based on the types of information you track.

Before You Begin

Decide what information to track and who relies on that information.

-   Gather business requirements.
    
-   Decide what types of fields to add.
    
-   Decide which user profiles require access to each field.
    

1.  From Setup, in the Quick Find box, enter Object Manager, and then select **Object Manager**.
2.  Find and select **Trailing Document Placeholder**.
3.  Click **Fields & Relationships**, then click **New**.
4.  Create a new custom field, then click **Save**.
