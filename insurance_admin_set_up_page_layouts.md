# Set Up Page Layouts and Record Pages for Insurance Coexistence

Customize the page layouts and record pages for Quote, Insurance Policy, and Claim so that users see only the fields, actions, and components that they need for Digital Insurance. Assign the page layouts and record pages to the Digital Insurance record types.

### Required Editions

Customize the page layouts for Quote, Insurance Policy, and Claim to add, remove, or reorder actions, buttons, fields, and sections on the record detail page. See [Customize Page Layouts with the Enhanced Page Layout Editor](https://help.salesforce.com/s/articleView?id=platform.layouts_customize_ple.htm&language=en_US&type=5). Then, assign the page layouts to the relevant record types. See [Assign Page Layouts to Profiles or Record Types](https://help.salesforce.com/s/articleView?id=platform.layouts_assigning.htm&language=en_US&type=5).

Customize the record pages for Quote, Insurance Policy, and Claim to add components for Digital Insurance workflows and remove components that are specific to the Insurance managed package or Insurance CRM. See [Create and Configure Lightning Experience Record Pages](https://help.salesforce.com/s/articleView?id=platform.lightning_app_builder_customize_lex_pages.htm&language=en_US&type=5). Then, assign the page layouts to the relevant record types and activate the pages. See [Activate Lightning Experience Record Pages](https://help.salesforce.com/s/articleView?id=platform.lightning_app_builder_customize_lex_pages_activate.htm&language=en_US&type=5).
