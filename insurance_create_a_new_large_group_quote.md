# Create a New Large Group Quote

To estimate how much a large group policy would cost, you use a quote.

All quotes begin as opportunities. To create a quote, start from an Opportunity record.

1.  Go to the Opportunities list view.
2.  Select the opportunity you want to create a quote for or create an opportunity.
3.  On the Quotes panel, click the carat or button for New Quote.
4.  Select Large Group Quote in the New Quote dialog box that appears, and then click **Next**.
    
    Note The quote record types and their associated pages and LWCs are customizable and configurable.
    
5.  On the New Quote window, enter a **Quote Name**.
6.  (Optional) Enter other quote information.
    
    Important Don't enter a value into the Primary Root Item. Entering a value into this field causes a conflict that doesn't let you modify the quote on the Quote object.
    
7.  Save the quote. When you click Save, you create a blank quote record. To configure the quote to meet your needs, add a plan.
