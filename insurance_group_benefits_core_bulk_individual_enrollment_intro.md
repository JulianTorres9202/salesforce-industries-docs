# Group Benefits Bulk and Individual Enrollment

The Insurance Group Census Lightning web component for the Salesforce data model shows group members, dependents. You can add, update, or delete group census members and dependents using the LWC. For the enrollment census type, you can also view plans information in the grid and configure the same using the Configure Plans modal. With the individual enrollment flow, individual group members can also view eligible plans, optional coverages, premiums, and contributions, and enroll themselves directly using the APIs or invocable actions.
