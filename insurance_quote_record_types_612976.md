# Quote Record Types

The **Type** field on the **Quote** Details tab helps you to differentiate between quotes created for new business and endorsement (mid-term adjustment) quotes.

[](https://help.salesforce.com/s?language=en_US)

## Record Type Options

[](https://help.salesforce.com/s?language=en_US)

-   **New Business**: Use this option when you create a new quote. Vlocity defaults to New Business when you create a quote from an Opportunity record.
    
-   **Endorsement**: Use this option when you create a mid-term adjustment quote. Vlocity defaults to Endorsement when you create a quote from a Policy (Asset) record.
