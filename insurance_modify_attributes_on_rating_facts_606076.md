# Modify Attributes on Rating Facts

You can make some changes to attributes in rating facts on a root product spec.

1.  Open the attribute category by clicking the carat to the left of the attribute category name.
2.  Change one or more of the following on each attribute as needed:
    
    -   **Rating**: Select or deselect if you want to change whether or not this attribute is used for rating the product.
        
    -   **Rating Input Mapping**: Enter or change a mapping for rating input for this attribute.
        
        You can view the attributes in JSON form by clicking **view JSON**.
