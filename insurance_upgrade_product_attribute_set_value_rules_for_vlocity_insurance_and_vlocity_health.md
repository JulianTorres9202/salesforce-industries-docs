# Upgrade Product Attribute Set Value Rules for Insurance

Fix any set value rules with incompatible or unsupported methods.

1.  From the App Launcher, open **Attribute Set Value Rules Upgrade**.
2.  If there are no set value rules that need to be fixed, a message is displayed:
    
    "There are no set value rules with incompatible or unsupported methods."
    
3.  If you have incompatible or unsupported expressions in your set value rules, a warning message appears above a list of Attribute Set Value Rules that must be corrected. You can make the necessary changes on this page.
