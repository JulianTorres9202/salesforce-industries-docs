# Configure Batch Process Job Definition for Commissions Management

Configure the batch job definition to use for producer commission processing.

### Required Editions

| User Permissions Needed |
| --- |
| To set up brokerage: | Insurance Brokerage User |

1.  In Setup, find and select **Brokerage**. Then, select **Commissions Management**.
2.  In the Batch Job Setting for Commission section, from the dropdown menu for Active Batch Job Definition, select the batch process job definition.
3.  Save your changes.
