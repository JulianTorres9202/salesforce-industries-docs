# Post-Upgrade Tasks for Insurance

To complete your upgrade, perform required post-upgrade tasks.

-   **[Activate and Deploy LWC OmniScripts](https://help.salesforce.com/s/articleView?id=ind.insurance_activate_and_deploy_an_lwc_omniscript_omniscripts.htm&language=en_US&type=5)**  
    Make OmniScripts available to Experience Sites, Lightning Pages, custom LWCs, and Lightning tabs by activating and deploying the OmniScript. If an error occurs during deployment the OmniScript will deactivate.
-   **[Create and Activate Lightning Record Pages for Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_create_and_activate_lightning_record_pages_asof242.htm&language=en_US&type=5)**  
    Create Lightning record pages that apply to your org, and activate them to make them available to users.
-   **[Edit Page Layout Assignments for Insurance](https://help.salesforce.com/s/articleView?id=ind.insurance_edit_page_layout_assignments_asof242.htm&language=en_US&type=5)**  
    After you upgrade, edit the page layout assignments for Product, Insured Item Spec, and Coverage Spec objects.
-   **[Verify and Update Required Permissions for Quoting](https://help.salesforce.com/s/articleView?id=ind.insurance_verify_required_quoting_permissions.htm&language=en_US&type=5)**  
    To view and create quotes, users must have access to certain information.
-   **[Field-Level Security Permission Set for Fields on the Quote Object](https://help.salesforce.com/s/articleView?id=ind.insurance_field_level_security_permission_set_for_fields_on_the_quote_object_2.htm&language=en_US&type=5)**  
    Provide your users with access to specific fields on the Quote object by creating a permission set.
-   **[Field-Level Security Permission Set for Insurance Industries Extension Objects and Fields in Upgrades](https://help.salesforce.com/s/articleView?id=ind.insurance_field_level_security_perm_set_for_iie_upgrades_asof242.htm&language=en_US&type=5)**  
    Provide your users with access to Insurance Industries Extension objects and fields by creating a permission set.
