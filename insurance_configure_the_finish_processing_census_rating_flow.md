# Configure the Finish Processing Census Rating Flow

Use this flow to finish processing a rating batch job, update the rating request status, and send a status notification.

1.  From Setup, in the Quick Find box, enter Flows, and then select **Flows**.
2.  Open the Finish Processing Census Rating Flow.
3.  Click **Save As**. Enter a label, API name, and description.
4.  Save and activate the flow.
    
    Note Make sure that you have only one clone of this flow template activated.
