# Manage Picklist Values in the Attribute Class Field of Vlocity Attribute Object for Insurance Releases Spring '21 Through Winter '25

Change the label and edit the API name of several values in the Attribute Class picklist on the Vlocity Attribute Object.

1.  From **Setup**, click the **Object Manager** tab.
2.  Click  **Vlocity Attribute**.
3.  Click **Fields & Relationships**.
4.  Click **Attribute Class**.
5.  In the **Values** section, verify that you have these values. If you don't, update them.
