# Configure Census Management for Group Benefits Quoting

Customize the Quote Census Lightning web component for user workflows and create rules that ensure users get the cleanest census data possible.

Important

The Quote Census Lightning web components insGroupCensus and insOSGroupCensus work with the Salesforce data model's Group Census and Group Census Member objects. To continue to use the Vlocity data model, use the Insurance and Health Managed package Lightning web components insCensus and inOSCensus.

-   **[Considerations for Using the Quote Census Lightning Web Component](https://help.salesforce.com/s/articleView?id=ind.insurance_considerations_for_using_the_quote_census_lightning_web_component.htm&language=en_US&type=5)**  
    The Quote Census Lightning web component provides insurance carrier administrators and brokers a straightforward workflow for managing census data in small and medium group quotes.
-   **[Configure Views of Group Benefits Census Data in Quotes](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_views_of_group_benefits_census_data_in_quotes.htm&language=en_US&type=5)**  
    Show insurance carrier administrators and brokers exactly the right vital statistics and details about uploaded census data. The Quote Census Lightning web component shows certain values by default. To customize views of census data, create field sets with standard and custom fields to suit your business needs.
-   **[Create Rules to Verify Uploaded Group Census Data](https://help.salesforce.com/s/articleView?id=ind.insurance_create_rules_to_verify_uploaded_group_census_data.htm&language=en_US&type=5)**  
    Use triggers and validation rules that identify incomplete and invalid census data. This helps insurance carrier administrators and brokers identify and resolve exceptions efficiently.
-   **[Configure the Quote Page and a Quote Workflow for Census Management](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_the_quote_page_and_a_quote_workflow_for_census_management.htm&language=en_US&type=5)**  
    Give users a straightforward way to incorporate census data directly into a quote or configure a quote workflow by incorporating the Quote Census Lightning web component into an OmniScript.
