# Create Catalog Categories and Subcategories

Organize and group products in your catalog by creating catalog categories and catalog subcategories. Assign products to more than one catalog category and subcategory.

Use the Categories tab for a hierarchical view of categories. To create a product category, follow these instructions:

1.  From the Product Catalog Management app's home page, click **Catalogs**.
2.  From the catalogs list view page, click **Insurance Claims Catalog**.
3.  Click **Categories**.
4.  Click **Create Category**.
5.  In the New Category window, enter these field values:
    1.  Enter a name and description for the catalog category. For example, Auto Insurance Claims and Insurance Claims Auto Products Category.
        
        The Catalog field is auto-populated with the catalog name.
        
    2.  Select a parent category for the catalog category.
        
        If the category you're creating is the first category in the hierarchy, leave the parent category field blank.
        
    3.  To show the catalog category as a navigational breadcrumb at runtime, select **Show In Menu**.
    4.  Enter the sort order of the category in the hierarchy.
        
        The sort order determines the order in which the categories are shown to users as part of the purchase flow.
        
    5.  Enter a category code.
6.  Save your changes.
7.  You can create more categories from the Categories tab. To update the category order, click **Sort**.
8.  To add a subcategory, click next to the category and click **Add Subcategory**.
    1.  Enter a name, description, code, and sort order for the subcategory.
        
        The Catalog and Parent Category fields populate automatically.
        
    2.  To show the subcategory as a navigational breadcrumb at runtime, select **Show In Menu**.
9.  Save your changes.
10.  To reorganize the order in which subcategories appear under a category, click next to a category and click **Update Subcategory Order**.
11.  Drag and drop the subcategories into the order you want.
12.  Save your changes.
