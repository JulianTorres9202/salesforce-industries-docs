Select the fields to show for a rate plan commission type.

| To set up brokerage: | Insurance Brokerage User |

1.  In Setup, find and select **Brokerage**.
2.  Select **Policy Management** and then select **Rate Plan**.
3.  In the Rate Plan Commission Configuration section, click **Add**.
4.  Select the rate plan commission type, calculation type, and the fields to show.
5.  Save your changes.
