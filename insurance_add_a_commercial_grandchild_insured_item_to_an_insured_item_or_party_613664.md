# Add a Commercial Grandchild Insured Item to an Insured Item or Party

If you're working with a multi-instance insurance product with grandchild insured items, you can add a grandchild item to an existing insured item on a commercial quote.

For example, you can add a building to a location on a quote.

1.  On the insured item you're adding to, use the dropdown to select the **Add** Item button.
    
    Example: If you are adding a building (a grandchild insured item) to a location (the insured item) on a commercial quote, you can select **Add Building**.
    
2.  In the **Add** Item window, fill out the appropriate information, then click **Save**.
