# Insurance View Census Data Permission Set

Give your agents access to just what they need to view census data.

[](https://help.salesforce.com/s?language=en_US)

## Access Provided by the Insurance View Census Data Permission Set

This permission set is delivered in the Insurance Industries Extension managed package and provides read-only access to the following objects:

[](https://help.salesforce.com/s?language=en_US)

-   Account
    
-   Attribute Assignment
    
-   Compiled Attribute Override
    
-   Contact
    
-   Contract
    
-   Contract Line Item
    
-   Group Census
    
-   Group Census Member
    
-   Group Census Member Plan
    
-   Group Class
    
-   Pricebook
    
-   Price List
    
-   Price List Entry
    
-   Producer Commission
    
-   Product
    
-   Product Child Item
    
-   Product Requirement
    

Additionally, this permission set grants read-only access to certain fields on some of the objects listed above.
