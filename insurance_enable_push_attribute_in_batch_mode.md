# Enable Push Attributes in Batch Mode

If your product spec is referenced by a large number of products and you're hitting Salesforce limits in push attribute, you can enable push attributes in batch mode.

1.  From **Setup**, in the **Quick Find** box, enter Custom Settings, and then select **Custom Settings**.
2.  Click **Manage** next to **Insurance Configuration Setup**, and then click **New**.
3.  For **Name**, enter PushAttributeInBatchMode.
4.  For **Setup Value**, enter true.
5.  Save the custom setting.
