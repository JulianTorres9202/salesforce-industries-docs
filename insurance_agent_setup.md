Before you begin, [Set Up Einstein Generative AI](https://help.salesforce.com/s/articleView?id=ai.generative_ai_enable.htm&language=en_US&type=5), turn on Agentforce, and assign permissions.

1.  From Setup, in the Quick Find box, enter Permission Sets and then select **Permission Set**.
2.  Create a new permission set to provide permission for agent chat.
3.  In the new permission set, click **Agent Access** and then click **Edit**.
4.  Select **Insurance Service Assistance** and click **Add** to add the agent to the Enabled Agents list and save your changes.
5.  Click Manage Assignments and assign the permission set to your users.
