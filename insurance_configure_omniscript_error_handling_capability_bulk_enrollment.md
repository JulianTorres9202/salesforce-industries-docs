# Download the Bulk Enrollment Configuration Solution

Before you configure the OmniScript and error handling for bulk enrollment and new hire enrollment processes, download the custom solution that's available as a static resource.

The solution includes:

-   An OmniScript that walks users through uploading census data for Group Benefits enrollments and that creates the related objects in the Salesforce data model.
-   An Apex class to generate the error CSV file, store the `asyncBulkRequestId` in the insurance contract, and check if the enrollment request has any failed jobs.
-   A FlexCard to generate the error CSV file for failed jobs.

1.  From Setup, in the Quick Find box, enter Static Resources and select **Static Resources**.
2.  Click **LargeGroupEnrollment\_Artifacts**.
3.  To download the file, click **View file**.
4.  When downloaded, extract its contents.
