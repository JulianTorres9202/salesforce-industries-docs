# Download and Import DataPack for Claim History

Install a DataPack that brings essential content into a Claim History tab out of the box.

1.  [](https://help.salesforce.com/s?language=en_US)From Setup, in the Quick Find box, enter Static Resources, and then select **Static Resources**.
2.  [](https://help.salesforce.com/s?language=en_US)Find and click **DP\_CUSTOM\_InsStoryObjectConfiguration**.
3.  [](https://help.salesforce.com/s?language=en_US)Right-click **View File** and select **Save Link As**.
4.  [](https://help.salesforce.com/s?language=en_US)Add .json to the end of the file name suggested, so the full name is DP\_CUSTOM\_InsStoryObjectConfiguration.json. Browse to a folder that's easy to find on your local computer, then click **Save**.
5.  [](https://help.salesforce.com/s?language=en_US)From the App Launcher, find and select **Vlocity Cards**.
6.  [](https://help.salesforce.com/s?language=en_US)Click **Import**.
7.  [](https://help.salesforce.com/s?language=en_US)Click **Browse** to find and select the file you downloaded, or drag it to the dialog box. Then click Next.
8.  [](https://help.salesforce.com/s?language=en_US)For **Select Items to Import**, make sure all the items are selected, then click Next.
9.  [](https://help.salesforce.com/s?language=en_US)For **Review Items to Import**, click Next, then click Done.
