# Insurance Create/Update Quote Permission Set

Give your agents access to just what they need to create and update quotes with the Insurance Create/Update Quote permission set.

[](https://help.salesforce.com/s?language=en_US)

## Access Provided by the Insurance Create/Update Quote Permission Set

This permission set is delivered in the Insurance managed package and provides Read access to the following objects:

[](https://help.salesforce.com/s?language=en_US)

-   Attribute Assignment
    
-   Compiled Attribute Override
    
-   Contact
    
-   Contract
    
-   Jurisdiction
    
-   Pricebook
    
-   Price List
    
-   Price List Entry
    
-   Producer Commission
    
-   Product
    
-   Product Child Item
    
-   Product Requirement
    
-   Vlocity Action
    
-   Vlocity State
    
-   Vlocity State Model
    
-   Vlocity State Model Version
    
-   Vlocity State Transition
    
-   Vlocity State Transition Rule
    

This permission set provides Read and Create access to the following object:

-   Trailing Document Placeholder
    

This permission set provides Read, Edit, and Create access to the following objects:

-   Account
    
-   Opportunity
    
-   Quote
    
-   Quote Item Relationship
    

The Insurance Create/Update Quote permission set also provide Read, Edit, Create, and Delete access to these objects:

-   Quote Line Item
    
-   Quote Line Item Group Class
    
-   Quote Pricing Adjustment
    

Additionally, this permission set grants access to certain fields on some of the objects listed above.
