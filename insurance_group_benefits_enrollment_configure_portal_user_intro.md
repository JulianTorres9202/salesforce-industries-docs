# Configure Enrollment for Portal User Creation

Learn how to extend Salesforce’s standard Group Enrollment process to also create portal users. This involves customizing the enrollment flow and setting up a batch job so census members enrolled in a group also get portal user accounts.

-   **[Configure the Salesforce Flow for Enrollment to Create Portal Users](https://help.salesforce.com/s/articleView?id=ind.insurance_group_benefits_core_enrollment_configure_flow_portal_user.htm&language=en_US&type=5)**  
    You can use the out-of-the-box (OOTB) Group Enrollment flow to enroll group census members and create insurance policies based on their plan and coverage selection in the group census member plan.
-   **[Configure the Batch Job to Create Portal Users](https://help.salesforce.com/s/articleView?id=ind.insurance_group_benefits_core_create_portal_users_batch_job.htm&language=en_US&type=5)**  
    The out-of-the-box Group Enrollment batch job references the Group Enrollment flow that’s shipped out of the box and can’t be modified. Perform this task to ensure the batch job uses the flow that’s customized to create portal users.
