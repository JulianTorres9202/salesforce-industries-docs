# Configure Preferences for Rate Plan Type

Select the fields to show for rate plan line items and rate plan commission type for a rate plan type.

### Required Editions

| User Permissions Needed |
| --- |
| To setup brokerage: | Insurance Brokerage User |

1.  In Setup, find and select **Brokerage**.
2.  Select **Policy Management** and then select **Rate Plan**.
3.  In the Rate Plan Type Configuration section, click **Add**.
4.  Select the rate plan type for which you want to configure the field visibility.
5.  In the Field Visibility section, select the fields to show on the rate plan line items.
6.  In the Rate Plan Commission Type section, select the applicable values for the Commission Type field.
7.  Save your work.
8.  To edit or delete the configuration for any rate plan type, click the respective action menu and select **Edit** or **Delete**.
9.  Update the configurations and save your changes.
