# Insurance Create/Update Group Census Data Permission Set

Give your agents access to just what they need to create and update group census data.

[](https://help.salesforce.com/s?language=en_US)

## Access Provided by the Insurance Create/Update Group Census Data Permission Set

This permission set is delivered in the Insurance managed package and provides Read access to the following objects:

[](https://help.salesforce.com/s?language=en_US)

-   Account
    
-   Attribute
    
-   Attribute Assignment
    
-   Attribute Category
    
-   Compiled Attribute Override
    
-   Contract
    
-   Group Census
    
-   Group Class
    
-   Group Class Contribution
    
-   Insurance Contract
    
-   Pricebook
    
-   Price List
    
-   Price List Entry
    
-   Product
    
-   Product Child Item
    
-   Product Requirement
    

This permission set provides Read, Edit, and Create access to the following objects:

-   Contact
    
-   Contract Group Plan
    

The Insurance Create/Update Group Census Data permission set also provide Read, Edit, Create, and Delete access to these objects:

-   Group Census Member
    
-   Group Census Member Plan
    

Additionally, this permission set grants access to certain fields on some of the objects listed above.
