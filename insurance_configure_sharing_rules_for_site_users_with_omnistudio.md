# Configure Sharing Rules for Site Users with OmniStudio

Sharing rules give your site users greater access by making automatic exceptions to your org-wide sharing settings.

Keep in mind that sharing settings in the Insurance Property & Casualty Application are for illustrative purposes and don't necessarily apply to your org.

[](https://help.salesforce.com/s?language=en_US)

1.  From Setup, use **Quick Find** to search for Sharing.
2.  Click **Sharing Settings**.
3.  Click **Edit**.
4.  Now set these object values for the default external access:
