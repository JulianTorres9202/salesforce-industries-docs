# Create an Attribute Definition with Currency and Percent Data Type

Create an attribute to describe the product properties. Create all attribute definitions that have the data type as currency or percent. For example, Deductible, Copay, and OOP Max have currency as data type and Coinsurance has percent as data type.

1.  Click the dropdown arrow for Product Catalog Management and select **Attribute Definitions**. Then, click **New**.
2.  Enter the attribute definition information and save your changes.
    
    In the Data Type field, for attribute definitions with Currency data type, select Currency, and for Percent data type, select Percent. For example, select Currency for Deductible and Percent for Coinsurance.
