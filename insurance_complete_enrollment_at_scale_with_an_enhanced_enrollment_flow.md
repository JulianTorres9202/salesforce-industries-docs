# Complete Enrollment at Scale With an Enhanced Enrollment Flow

Enroll up to 10,000 members at a time with up to 30 distinct plan choices for each member by using the enhanced Enrollment flow. With duplicate policy checks, avoid creating another policy for the same member within the same plan category and save time.

**Where**: This change applies to Lightning Experience in Professional, Enterprise, and Unlimited editions with the Insurance Industries managed package and the Insurance Industries Extension managed package.

**How**: Deactivate all versions of this flow that you installed with the Winter '24 release. Then, update the Enrollment flow from the Spring ’24 version and configure it in your batch configuration for the large group enrollment process.
