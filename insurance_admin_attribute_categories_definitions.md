# Create Attribute Categories and Attribute Definitions for Digital Insurance

Create attributes to capture the characteristics or properties of products. For example, limit and deductible are attributes of coverage whereas age, gender, first name are attributes of a driver.

1.  Create picklists to allow you to define multiple values for each attribute. For example, marital status for a person or limit for coverage. See [Manage Attribute Picklists in Revenue Cloud](https://help.salesforce.com/s/articleView?id=ind.product_catalog_picklists.htm&language=en_US&type=5#product_catalog_picklists).
2.  Create the attributes for the products you offer. See [Create Product Attributes in Revenue Cloud](https://help.salesforce.com/s/articleView?id=ind.product_catalog_dyn_create_attributes.htm&language=en_US&type=5).
3.  If you want to further organize attributes, create attribute categories. For example, age, gender, marital status, first name can be grouped together in attribute category “Personal Information.” See [Create Attribute Categories in Revenue Cloud](https://help.salesforce.com/s/articleView?id=ind.product_catalog_dyn_create_attribute_categories.htm&language=en_US&type=5#product_catalog_dyn_create_attribute_categories).
