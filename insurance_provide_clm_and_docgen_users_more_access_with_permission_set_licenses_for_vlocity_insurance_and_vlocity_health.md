# Provide CLM and DocGen Users More Access with Permission Set Licenses for Vlocity Insurance

Starting Spring ’22, new customers can use permission set licenses to incrementally extend the functionality of user licenses. CLM, and DocGen offer permission set licenses for Vlocity Insurance. With permission set licenses, you can assign more permissions to users than what their user license supports.

Permission set licenses are available only for new customers who have purchased the new Salesforce SKUs post 17 May 2022, and installed the Spring ’22 package. Existing customers upgrading to Spring '22 continue using managed package licenses. However, existing customers upgrading to Spring '22 can migrate from managed package license to permission set licenses during contract renewal.

**How:** Install the Spring ’22 Vlocity Insurance package. From Setup, in the Quick Find box, enter Company Information, and then select Company Information. Go to the permission set licenses related list, click the name of the permission set license, and then assign users. View users to see and modify current assignments.
