# Add Fields to Object Field Sets for Insurance Industries Extension Releases Spring '21 Through Winter '25

Add fields to field sets on several objects used with Insurance Industries Extension.

1.  From **Setup**, click the **Object Manager** tab.
2.  Click an object from the list below.
3.  Click **Field Sets**.
4.  Click a Field Set Field Label listed for that object.
5.  Find the fields listed in the top panel down, then drag them into the **In This Field Set** area.
6.  Click **Save**.
7.  Repeat these steps for each object, field set, and field in this list.
    
    | Object | Field Set Field Label | Fields | Release Added |
    | --- | --- | --- | --- |
    | Insurance Policy | GetPolicyDetails | Policy | Summer '21 |
    | Policy Name | Summer '21 |
    | Producer | Summer '21 |
    | Ins\_GetPolicyDetails | Policy | Summer '21 |
    | Policy Name | Summer '21 |
    | Producer | Summer '21 |
    | Production Code Id | Summer '21 |
