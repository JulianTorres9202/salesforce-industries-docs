# Configure Sharing Rules for Site Users

Sharing rules give your site users greater access by making automatic exceptions to your org-wide sharing settings.

1.  From Setup, use **Quick Find** to search for sharing.
2.  Click **Sharing Settings**.
3.  Click **Edit**.
4.  Now set these object values for the default external access:
