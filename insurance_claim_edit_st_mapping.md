# Edit Static to Dynamic Status Mapping

You can add a new dynamic value to an already mapped static value or change the default dynamic status value.

1.  From Setup, in the Quick Find box, enter Insurance Claim Services, and then select **Configure Picklist Mapping**.
2.  Click **Edit Picklist Mapping**.
3.  To edit a picklist mapping, select the static value you want to modify.
4.  Click **Next**.
5.  Add or remove dynamic values for the selected static value by using the arrow buttons.
6.  Click **Next**.
7.  If you select multiple dynamic values, select one value to be the default.
8.  Save your changes.
