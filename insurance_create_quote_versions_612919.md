# Create Quote Versions

To generate a new version of a quote, create a version from the Quote record.

1.  Make changes to an existing quote by adding, removing, or changing coverages.
2.  Click **Rate Now** to save the updates to the quote.
3.  Click **Create Version**.
4.  Enter a unique **Quote Name**.
5.  Click **Create**.
