# Server-Side Document Generation

Use server-side document generation for automatic or asynchronous invocation, and for documents with lots of token processing, tabulation, or line items.

**Where:** Server-side document generation is available as a pilot feature in Vlocity Contract Lifecycle Management for Summer '21 Vlocity Insurance. To request consideration for the pilot program, contact Salesforce Support.

**Why:** Client-side document generation has limitations: it's best for ad hoc generation of lightweight documents that don't overload web browser resources. Server-side document generation overcomes these limitations. It supports automatic or asynchronous invocation, and it can handle documents with lots of token processing, tabulation, or line items.

**How:** A sample OmniScript is provided that can perform server-side document generation based on any Microsoft Word or PowerPoint template with data from any Salesforce object.
