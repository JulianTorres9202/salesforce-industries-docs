# Add the Push Attribute Action to the Product Page

If you update a child spec, you can use the Push Attribute action to push your changes out to the instances of the child spec in other products.

1.  From Setup, in the Quick Find box, enter Object Manager. Then find and click **Product**.
2.  Go to **Page Layouts** and select the layout you're using. For example, select **Product Layout (Vlocity 108)**.
3.  Under **Salesforce Mobile and Lightning Experience Actions**, click **override the predefined actions**.
4.  In the top panel, select **Mobile & Lightning Actions**.
5.  Find the **Push Attribute** action and drag it to the **Salesforce Mobile and Lightning Experience Actions** panel.
6.  Back up in the top panel, click **Save**.
