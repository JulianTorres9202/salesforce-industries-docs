# Insurance Create/Update Enrollment Permission Set

Give your agents access to just what they need to create and update enrollment information.

[](https://help.salesforce.com/s?language=en_US)

## Access Provided by the Insurance Create/Update Enrollment Permission Set

This permission set is delivered in the Insurance Industries Extension managed package and provides Read access to the following objects:

-   Attribute Assignment
    
-   Compiled Attribute Override
    
-   Pricebook
    
-   Price List
    
-   Price List Entry
    
-   Producer Commission
    
-   Product
    
-   Product Child Item
    
-   Product Requirement
    

This permission set provides Read, Edit, and Create access to the following objects:

[](https://help.salesforce.com/s?language=en_US)

-   Insurance Policy
    
-   Insurance Policy Coverage
    
-   Insurance Policy Participant
    

Additionally, this permission set grants access to certain fields on some of the objects listed above.
