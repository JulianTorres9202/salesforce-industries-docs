# Update Session Settings for the Insurance Installation in Government Cloud Orgs

To automatically add core picklist values to your Government Cloud org using the Installation Assistant, you must disable the **Lock sessions to the IP address from which they originated** setting.

1.  From Setup, enter Session Settings in the Quick Find box, then select **Session Settings**.
2.  Deselect **Lock sessions to the IP address from which they originated**.
3.  Click **Save**.
