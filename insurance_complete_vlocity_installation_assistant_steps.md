# Complete Vlocity Installation Assistant Steps

To complete the installation and the configuration of Insurance do all the tasks marked New Installation Only and all the tasks not specially marked.

Note If you're installing the Insurance package in a Government Cloud org, disable the **Lock sessions to the IP address from which they originated** setting prior to using the Vlocity Installation Assistant. To learn how to disable this setting, see [Update Session Settings for the Insurance Installation in Government Cloud Orgs](https://help.salesforce.com/s/articleView?id=ind.insurance_update_session_settings_for_vlocity_insurance_and_vlocity_health_installation_in_government_cloud_orgs.htm&language=en_US&type=5 "To automatically add core picklist values to your Government Cloud org using the Installation Assistant, you must disable the Lock sessions to the IP address from which they originated setting.").

1.  From the App Launcher, find and select **Vlocity Installation Assistant**.
2.  Click the **Assistant** tab.
3.  Click **Reload Steps** to see all the configuration tasks.
4.  Beginning with the first task, follow the instructions to complete each task in the Assistant. You can complete some tasks automatically. For automatic tasks, click the **Complete All** button at the top to do all the tasks at once. To do automatic steps one at a time, click **Complete Step** next to each individual step. You must complete some steps manually. For manual tasks:
    1.  Click **Show Instructions** next to the task.
    2.  Do the task in your Insurance org, in a new tab if possible.
    3.  Go back to the Vlocity Installation Assistant, and on the Instructions window click **Mark Complete**
5.  After you complete all the steps, uninstall the Vlocity Installation Assistant. See [Install the Vlocity Installation Assistant](https://help.salesforce.com/s/articleView?id=ind.insurance_install_the_installation_assistant_asof242.htm&language=en_US&type=5#insurance_install_the_installation_assistant_asof242 "To make it easier to configure Insurance after installing or upgrading, install the unmanaged package that contains the Installation Assistant.").
