# Create and Activate Lightning Record Pages for Insurance

Create Lightning record pages that apply to your org, and activate them to make them available to users.

If you previous installed the 890.210 or 890.344 unmanaged package or have done this step before, you're all set. If not, create and activate Lightning record pages manually.
