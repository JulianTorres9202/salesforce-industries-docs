# Create a Group Class

Group classes are groups of members who all receive similar benefits. Examples of group classes include Executives, Managers, and Directors.

| To create a group class | Digital Insurance Group Census Quote Contract Management, Digital Insurance Group Census Quote Contract Management Partner Community |

1.  From the App Launcher, find and select **Group Class**.
2.  Click **New**.
3.  Enter these details:
    
    -   Specify a group class name. For example, Executives.
    -   Select an account associated with this group class.
    
4.  Save your changes.
