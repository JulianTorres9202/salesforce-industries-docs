Manage all the configurations related to commission management.

-   **[Configure Commission Statement Line Item Type](https://help.salesforce.com/s/articleView?id=ind.insurance_brokerage_setup_commission_statement_line_item_type.htm&language=en_US&type=5)**  
    Calculate producer commissions by mapping the commission statement line item commission type to the producer split arrangement line item split arrangement type.
-   **[Map Commission Statement Status](https://help.salesforce.com/s/articleView?id=ind.insurance_brokerage_setup_commission_statement_status.htm&language=en_US&type=5)**  
    Map the system status of the commission statement to the customer-facing status of the commission statement.
-   **[Configure Commission Statement Line Item Status](https://help.salesforce.com/s/articleView?id=ind.insurance_brokerage_setup_commission_statement_line_item_status.htm&language=en_US&type=5)**

    Define which statuses are eligible for reprocessing the commission statement line item when the processing fails.
-   **[Configure Matching Criteria for an Insurance Policy](https://help.salesforce.com/s/articleView?id=ind.insurance_brokerage_setup_matching_criteria.htm&language=en_US&type=5)**  
    Identify the policy IDs for commission processing by mapping the insurance policy fields with the commission statement line item fields.
-   **[Configure Invalid Statuses for Producer Commissions](https://help.salesforce.com/s/articleView?id=ind.insurance_brokerage_setup_commission_invalid_producer_status.htm&language=en_US&type=5)**  
    Define which producer commission status is invalid.
-   **[Configure Batch Process Job Definition for Commissions Management](https://help.salesforce.com/s/articleView?id=ind.insurance_brokerage_setup_commission_batch_process_job.htm&language=en_US&type=5)**  
    Configure the batch job definition to use for producer commission processing.
