# Create Custom Numbering

Insurance generates a unique number for each new insurance policy, claim, census, and other items as they're created.

You can create a custom numbering system that includes specific information, such as versions, years, and product codes in your numbers.

To set up a custom numbering system:

1.  [Set Up the Global Auto-Number Generator Function](https://help.salesforce.com/s/articleView?id=ind.insurance_set_up_the_global_auto_number_generator_function_608778.htm&language=en_US&type=5 "If you plan to use Vlocity's auto-number generator function as part of your custom numbers, you must add it to Custom Settings.")
2.  [Set Up the Formula for Custom Numbers](https://help.salesforce.com/s/articleView?id=ind.insurance_set_up_the_formula_for_custom_numbers_608853.htm&language=en_US&type=5 "A custom numbering system can include specific information and you can establish a custom formula for that system.")
3.  [Add the Custom Number Generator to an OmniScript](https://help.salesforce.com/s/articleView?id=ind.insurance_add_the_custom_number_generator_to_an_omniscript_608900.htm&language=en_US&type=5 "Use an OmniScript to add the custom policy numbering to your policy creation experience.")
4.  [Use a Data Mapper to Auto-Number a Census](https://help.salesforce.com/s/articleView?id=ind.insurance_use_a_data_mapper_to_auto_number_a_census.htm&language=en_US&type=5 "You can also use an Omnistudio Data Mapper to auto-number census records so that they number sequentially, specific to the census object, by using a formula.")

-   **[Set Up the Global Auto-Number Generator Function](https://help.salesforce.com/s/articleView?id=ind.insurance_set_up_the_global_auto_number_generator_function_608778.htm&language=en_US&type=5)**  
    If you plan to use Vlocity's auto-number generator function as part of your custom numbers, you must add it to Custom Settings.
-   **[Set Up the Formula for Custom Numbers](https://help.salesforce.com/s/articleView?id=ind.insurance_set_up_the_formula_for_custom_numbers_608853.htm&language=en_US&type=5)**  
    A custom numbering system can include specific information and you can establish a custom formula for that system.
-   **[Add the Custom Number Generator to an OmniScript](https://help.salesforce.com/s/articleView?id=ind.insurance_add_the_custom_number_generator_to_an_omniscript_608900.htm&language=en_US&type=5)**  
    Use an OmniScript to add the custom policy numbering to your policy creation experience.
-   **[Use a Data Mapper to Auto-Number a Census](https://help.salesforce.com/s/articleView?id=ind.insurance_use_a_data_mapper_to_auto_number_a_census.htm&language=en_US&type=5)**  
    You can also use an Omnistudio Data Mapper to auto-number census records so that they number sequentially, specific to the census object, by using a formula.
