# Set the Default Value of the Premium Frequency Field on the Insurance Policy Object

Set the default value of the Insurance Policy object's Premium Frequency picklist to Annually.

1.  From **Setup**, click the **Object Manager** tab.
2.  Click **Insurance Policy**.
3.  Click **Fields & Relationships**.
4.  Click **Premium Frequency**.
5.  In the **Premium Frequency Picklist Values** section, click **Edit** next to the value of **Annually**.
6.  Select the **Default** checkbox.
7.  Click **Save**.
