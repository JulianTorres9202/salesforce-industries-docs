# Debug Attribute and Attribute Value Rules

Debug mode gives you insight into attributes and attribute rules for a quote.

Before You Begin

You must enable Debug Mode on the Quote UI before you can use the Attribute and Attribute Value Rules Debug Log.

-   **Enable Debug Mode** and **Show Actions** using the steps in [Enable Quote UI Configuration Options](https://help.salesforce.com/s/articleView?id=ind.insurance_enable_quote_ui_configuration_options_611168.htm&language=en_US&type=5 "You can choose several Quote UI configuration options based on your business needs.")
    
-   Verify that the **Debug Mode** button is visible on the Quote LWC page.
    

1.  Click **Debug Mode**.
    
    The button label changes to **Production Mode**.
    
2.  In debug mode, you can open a root product, an insured item, or a coverage and see its attribute rules in a table.
3.  Click **Production Mode** to turn debug mode off.
