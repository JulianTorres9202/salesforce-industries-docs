# insRecordHeader LWC

This component displays record data as a header.

This component works on the desktop and is mobile responsive.

[](https://help.salesforce.com/s?language=en_US)

[](https://help.salesforce.com/s?language=en_US)Here's what it looks like:

The **insRecordHeader** LWC displays:

1.  Custom image for record address
    
    This is an optional configuration.
    
2.  Custom icon for record
    
    This is an optional configuration and by default displays the initials of the record name.
    
3.  Record name
    
4.  Record details
    
    Users can customize the details that are displayed here.
    
5.  Record status
    
    This is an optional configuration.
    
6.  Weather data for the record address
    
    This is an optional configuration.
    

[](https://help.salesforce.com/s?language=en_US)

## What Users Can Do on this LWC

With the **insRecordHeader** LWC, users can:

-   View record details
    
-   View record status
    
-   Optionally view custom images and weather data
    

[](https://help.salesforce.com/s?language=en_US)

## What this LWC Does

The **insRecordHeader** LWC displays record information as a header in the flexcard.

[](https://help.salesforce.com/s?language=en_US)

## Services Called by this LWC

The **insRecordHeader** LWC doesn't call any services.

[](https://help.salesforce.com/s?language=en_US)

## Other LWCs Used by this Component

The **insRecordHeader** LWC doesn't have any LWCs inside it.
