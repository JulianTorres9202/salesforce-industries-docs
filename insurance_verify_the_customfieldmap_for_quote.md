# Verify the CustomFieldMap for Quote

After you install the FSCCustomFieldMap file, verify the custom field mappings for the Quote object.

1.  From the App Launcher, open **Field Mapper**.
2.  In **Source Object**, select **Insurance Policy<InsurancePolicy>**.
3.  In **Destination Object**, select **Quote<Quote>**.
4.  Click **Load Settings**.
5.  Verify the mapped fields listed on the **Fields Map** panel.
    
    Add other custom field mappings as needed.
