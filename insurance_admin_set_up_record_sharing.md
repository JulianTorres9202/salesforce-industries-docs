# Configure Record Sharing for Salesforce Pricing

For runtime users to access the data created by product designers or catalog admins, set up record sharing for all the Pricing entities.

For runtime setup, from Setup, in the Quick Find box, find and select **Sharing Settings**. Set up record sharing by using OWD or Sharing Rules for the following entities to suit your security requirements.

-   Price Book
-   Attribute Based Adjustment Rule
-   Cost Book
-   Decision Matrix
-   Derived Price
-   Expression Set
-   Price Adjustment Schedule
