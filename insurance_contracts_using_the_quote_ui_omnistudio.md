# Contracts Using the Quote UI

You can create a contract using the quote UI, which is built with Lightning Web Components.

1.  Navigate to the quote for which you want to create the contract.
2.  In **Actions**, click **Create Contract**.
3.  Review the group information displayed and click **Next**.
4.  The proposal is generated. This will take a couple of minutes.
    
    You can now download a copy of the contract or email the contract to the email ID listed in the account associated with the opportunity.
