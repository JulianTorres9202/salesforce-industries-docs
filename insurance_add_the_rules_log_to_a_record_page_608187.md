# Add the Rules Log to a Record Page

Salesforce admins can add the Rules Log Lightning web component to quotes, assets (policies), insurance policies, and claims.

1.  From the management settings for an object, go to **Lightning Record Pages**.
2.  On the left pane, find the standard **Accordion** component and drag it onto the record page.
    
    Tip We designed these pages with the Accordion component on the right side of the Lightning Record Pages. You can also use the Tab component if you want to.
    
3.  Back on the left pane, find the **Vlocity Rules Log** custom component.
4.  Drag the **Vlocity Rules Log** component onto the **Accordion**.
5.  (Optional) Rename the component Rules Log or another name of your choice.
6.  For the Quote page, click the **Quote LWC** tab, then click inside the tab to select the Vlocity Insurance Quote component.
7.  In the properties pane, select the Enable Debug Mode configuration option.
8.  Click **Save**.
