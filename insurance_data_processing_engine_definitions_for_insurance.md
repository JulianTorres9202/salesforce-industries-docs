# Data Processing Engine Definitions for Insurance

Data Processing Engine helps you transform data that’s available in your Salesforce org and writes back the transformation results as new or updated records. You can transform the data for standard and custom objects using Data Processing Engine definitions. After you set up a Data Processing Engine, you can use a flow to run it.

-   **[Use Data Processing Engine Templates](https://help.salesforce.com/s/articleView?id=ind.insurance_use_data_processing_engine_templates.htm&language=en_US&type=5)**  
    The prebuilt data processing engine templates for insurance allow you to create datasets that improve sales and extend customer outreach.
-   **[Clone Data Processing Engine Templates](https://help.salesforce.com/s/articleView?id=ind.insurance_clone_data_processing_engine_templates.htm&language=en_US&type=5)**  
    You can clone the predefined Data Processing Engine templates and customize them according to your business needs if the predefined templates don’t meet your requirements.
-   **[Activate a Data Processing Engine Definition](https://help.salesforce.com/s/articleView?id=ind.insurance_activate_a_data_processing_engine_definition.htm&language=en_US&type=5)**  
    After you’ve cloned the Data Processing Engine definitions, you must activate them.
