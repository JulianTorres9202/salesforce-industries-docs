# State Models for Property & Casualty Application Suite

All Property & Casualty Application Suite lines of business use state models to underpin their workflow and claim (underwriting) rules. Vlocity Insurance uses state models to transition a quote, a policy, or a claim from one state to another when a rule evaluates to true.

To learn what a state model is and how to create your own, see [Workflow for Creating a State Model, States, and Transitions](https://help.salesforce.com/s/articleView?id=ind.os_workflow_for_creating_a_state_modelstatesand_transitions_180716.htm&language=en_US&type=5).

The Property & Casualty Application Suite includes the following state models:

-   Claim Processing
    
-   Insurance Quote Underwriting
    
-   Insurance Renewals
