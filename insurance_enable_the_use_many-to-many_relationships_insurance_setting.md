# Enable the Use Many-to-Many Relationships Insurance Setting

Give insurance agents the tools to handle many-to-many relationships.

1.  From Setup, in the Quick Find box, enter Insurance Settings, and then select **Insurance Settings**.
2.  Enable the **Use Many-to-Many Relationships** setting.
