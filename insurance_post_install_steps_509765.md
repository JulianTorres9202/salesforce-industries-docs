# Post-Install Steps

Before exploring the Property & Casualty Application org, you need to complete a few configuration tasks to make sure you don't run into any glitches while you're doing your thing. This section walks you through all the post-deployment steps that you must complete.

-   [Configure the Self-Service Portal](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_the_self_service_portal_509794.htm&language=en_US&type=5 "Any member sites that you've created must be updated and also enabled for standard external profiles so that your policyholders and agents can access them.")
    
    [](https://help.salesforce.com/s?language=en_US)Any member sites that you've created must be updated and also enabled for standard external profiles so that your policyholders and agents can access them.
    
-   [Configure Sharing Rules for Site Users](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_sharing_rules_for_site_users_509838.htm&language=en_US&type=5 "Sharing rules give your site users greater access by making automatic exceptions to your org-wide sharing settings.")
    
    [](https://help.salesforce.com/s?language=en_US)Sharing rules give your site users greater access by making automatic exceptions to your org-wide sharing settings.
    
-   [Deactivate and Activate OmniScripts](https://help.salesforce.com/s/articleView?id=ind.insurance_deactivate_and_activate_omniscripts_509999.htm&language=en_US&type=5 "Log in to your org and then deactivate and activate OmniScripts so they can run as intended.")
    
    [](https://help.salesforce.com/s?language=en_US)Log in to your org and then deactivate and activate OmniScripts so they can run as intended.
    
-   [Configure Renewal Notifications](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_renewal_notifications_510048.htm&language=en_US&type=5 "You can send email notifications for policies that are up for renewal. When sending this email from Salesforce, you can select the organization-wide email address for the email’s From field and notifications are sent to email addresses listed in the member's Account record.")
    
    [](https://help.salesforce.com/s?language=en_US)You can send email notifications for policies that are up for renewal. When sending this email from Salesforce, you can select the organization-wide email address for the email’s **From** field and notifications are sent to email addresses listed in the member's Account record.
    
-   [Select Images for Property Claims](https://help.salesforce.com/s/articleView?id=ind.insurance_select_images_for_property_claims_510104.htm&language=en_US&type=5 "If images for the Property claims FNOL process aren't displaying correctly, configure them in the OmniScript.")
    
    [](https://help.salesforce.com/s?language=en_US)If images for the Property claims FNOL process aren't displaying correctly, configure them in the OmniScript.
    
-   [Set up the Google Map API Key](https://help.salesforce.com/s/articleView?id=ind.insurance_set_up_the_google_map_api_key_510242.htm&language=en_US&type=5 "Use Google Map APIs to automatically populate address fields in the Property and Casualty application.")(Optional)
    
    [](https://help.salesforce.com/s?language=en_US)Did you know that the Property and Casualty application supports Google Map APIs to automatically populate address fields? By default, this functionality is disabled for all OmniScripts.
    

-   **[Configure the Self-Service Portal](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_the_self_service_portal_509794.htm&language=en_US&type=5)**  
    Any member sites that you've created must be updated and also enabled for standard external profiles so that your policyholders and agents can access them.
-   **[Configure Sharing Rules for Site Users](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_sharing_rules_for_site_users_509838.htm&language=en_US&type=5)**  
    Sharing rules give your site users greater access by making automatic exceptions to your org-wide sharing settings.
-   **[Deactivate and Activate OmniScripts](https://help.salesforce.com/s/articleView?id=ind.insurance_deactivate_and_activate_omniscripts_509999.htm&language=en_US&type=5)**  
    Log in to your org and then deactivate and activate OmniScripts so they can run as intended.
-   **[Configure Renewal Notifications](https://help.salesforce.com/s/articleView?id=ind.insurance_configure_renewal_notifications_510048.htm&language=en_US&type=5)**  
    You can send email notifications for policies that are up for renewal. When sending this email from Salesforce, you can select the organization-wide email address for the email’s **From** field and notifications are sent to email addresses listed in the member's Account record.
-   **[Select Images for Property Claims](https://help.salesforce.com/s/articleView?id=ind.insurance_select_images_for_property_claims_510104.htm&language=en_US&type=5)**  
    If images for the Property claims FNOL process aren't displaying correctly, configure them in the OmniScript.
-   **[Set up the Google Map API Key](https://help.salesforce.com/s/articleView?id=ind.insurance_set_up_the_google_map_api_key_510242.htm&language=en_US&type=5)**  
    Use Google Map APIs to automatically populate address fields in the Property and Casualty application.
