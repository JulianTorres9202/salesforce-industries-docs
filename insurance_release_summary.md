# Insurance Release Summary

Get release summaries for Insurance minor releases delivered each Winter, Spring, and Summer.

Insurance patch releases are cumulative, meaning each patch includes all previous fixes. You can use the latest available release for new installations or upgrades.

Automatic upgrades are required starting in Spring '24. For more information, see [Salesforce Industries Automatic Upgrade](https://help.salesforce.com/s/articleView?id=000391836&language=en_US&type=1).

Note Package installation links are only available for the current release. To install older Insurance packages, request download links from Salesforce Support.
