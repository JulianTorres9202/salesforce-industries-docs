# Delete Plan Benefits

Delete a single benefit record or multiple benefit records.

### Required Editions

| User Permissions Needed |
| --- |
| To delete plan benefits: | Insurance Policy Management User |

1.  Go to the Insurance policy record page and select the **Plan Benefits** tab.
2.  To delete a single benefit, click the action menu of the benefit record that you want to delete and select **Delete**.
3.  Review the warning message and click **Delete**.
4.  To delete all benefits, click the action menu and select **Delete All**.
5.  Review the warning message and click **Delete**.
