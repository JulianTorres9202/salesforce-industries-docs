# Add a Product to a Quote

Add a product to a quote to start configuring it. You must add a product to a quote before you can add insured items or parties.

1.  From the Quote list page, open the quote you want to add a product to.
2.  Click **Add Product** on the **Quote LWC** tab.
    
    Note Quote LWC is the default tab name for the quote Lightning web component. Your admin can configure this tab name.
    
3.  Select the product (or products) you want to add to the quote, then click **Add**.
