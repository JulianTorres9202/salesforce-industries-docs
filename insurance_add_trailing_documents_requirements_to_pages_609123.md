# Add Trailing Documents Requirements to Pages

To meet internal or external compliance requirements, configure an object page to require a trailing document.

1.  Go to the <Object> detail view, go to the gear menu, and select Edit Page.
2.  Drag and drop the Vlocity Required Documents onto the <Object> detail page and specify ins-required-documents as the layout name.
3.  If no categories exist for the <Object>, you can create a default category.
4.  Go to an <Object> page, upload files into a category, then verify that the files uploaded correctly.
