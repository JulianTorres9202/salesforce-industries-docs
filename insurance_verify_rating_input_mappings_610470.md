# Verify Rating Input Mappings

After you map rating inputs to attributes, make sure input variables look right on the product spec.

[](https://help.salesforce.com/s?language=en_US)Go to the root product spec and click the Simulate tab. For all inputs that you mapped from an expression set, the **Input Mapping** displays the input variable you mapped. Default values defined for these inputs appear in the Simulate Values column.

If you mapped inputs from an Integration Procedure, they don't appear in the simulator. You must type in both the input variable and the value you want to use for the simulation.

[](https://help.salesforce.com/s?language=en_US)Note

If you have Integration Procedures from earlier releases, you can use calculation matrices and calculation procedures instead of lookup tables and expression sets.
