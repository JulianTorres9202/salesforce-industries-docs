# Insurance Winter '22 Release Notes

These release notes provide details about the changes in the Winter '22 release of Insurance.

-   **[Manage Claims and Policies Effectively with Insurance Claims and Policy Analytics](https://help.salesforce.com/s/articleView?id=ind.insurance_manage_claims_and_policies_effectively_with_insurance_claims_and_policy_analytics.htm&language=en_US&type=5)**  
    Use the Insurance Claims and Policy Analytics app to get actionable insights into claims management and policy administration. Claim handlers and producers can prioritize tasks, claims, quotes, and policies that need immediate attention to improve customer satisfaction. Team leaders can assess their team’s performance based on key metrics for claims and tasks.
-   **[Provide Users More Access with Vlocity Insurance Permission Set Licenses](https://help.salesforce.com/s/articleView?id=ind.insurance_provide_users_more_access_with_vlocity_insurance_permission_set_licenses.htm&language=en_US&type=5)**  
    Permission set licenses extend the functionality of user licenses. With permission set licenses, you can assign more permissions to users than what their user license supports.
-   **[Create Inbound Quote from An External System](https://help.salesforce.com/s/articleView?id=ind.insurance_create_inbound_quote_from_an_external_system.htm&language=en_US&type=5)**  
    Use `InsQuoteService:createUpdateQuoteFromExternal` service to create a quote or update an existing quote with new information through an inbound API call from an external policy admin system.
-   **[Data Model Updates for Vlocity Insurance Winter '22](https://help.salesforce.com/s/articleView?id=ind.insurance_data_model_updates_for_vlocity_insurance_winter__22.htm&language=en_US&type=5)**  
    This section provides a list of new data model objects and enhanced data model objects for this release.
-   **[Vlocity Insurance Application Suite Winter '22 Minor Release 234.1](https://help.salesforce.com/s/articleView?id=ind.insurance_vlocity_insurance_application_suite_winter22minor_release_234_1_2694.htm&language=en_US&type=5)**  
    This section describes key fixes and enhancements for the Vlocity Insurance Application Suite in this minor release.
