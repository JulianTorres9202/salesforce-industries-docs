# Set Up Trailing Documents Requirements in Communities

Increase customer engagement and speed up processing with trailing documents in Communities. Let customers and brokers upload supporting documents associated with objects such as quotes, policies, and claims.

1.  In the Community Builder, go to the <Object> detail page.
2.  Drag and drop the Vlocity Required Document component, then publish the changes.
    
    For example, you can add Vlocity Required Documents to the Quote Detail page in the Broker portal.
    
3.  Log in to the Community and go to the Quote detail page.
4.  Upload files into a category, then verify that the files uploaded correctly.
