# Clone a Component's FlexCard

When you want to customize a component for the Vlocity Insurance Broker or Policyholder Experience site, the first task is to clone the component's FlexCard.

The highlighted area in this flow shows where you are in the process of customizing Experience components:

Tip

If you plan to customize a bunch of Experience components, you can clone them all first, then move on to the next step to customize them.

1.  From the FlexCard Designer list view, open the version of the card you want to clone.
2.  Along the top right above the card, click **Clone**.
3.  On the **Clone Card** modal, enter a new name for the card.
4.  (Optional) Make changes to any other fields on the modal that you want.
5.  Click **Clone**.

[](https://help.salesforce.com/s?language=en_US)

[](https://help.salesforce.com/s?language=en_US)The system opens the new card.

[](https://help.salesforce.com/s?language=en_US)This card includes all the configurations of the original you cloned, including calls to data sources such as LWCs and Omnistudio Data Mappers.
