# Add a Price Book Entry

After you create a product, add it to a price book.

Note

This is a Salesforce requirement. Vlocity Health and Vlocity Insurance do not use the price book entry to price these products.

1.  Go to the **Related** tab and scroll down to **Price Books**.
2.  Click **Add Standard Price**.
3.  Leave the **Product**, **Active**, and **Price Book** fields as is.
4.  Enter a **List Price**. This can be any value because this value is not used by Vlocity.
