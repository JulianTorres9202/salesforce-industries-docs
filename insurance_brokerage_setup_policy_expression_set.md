# Configure Expression Sets for Policy Management

Configure expression sets to calculate the fee, premium, or premium equivalent, and for revenue forecasting.

| User Permissions Needed |
| --- |
| To set up brokerage: | Insurance Brokerage User |

1.  In Setup, find and select **Brokerage**. Select **Policy Management**, and then select **Premium Calculation**.
2.  In the Insurance Policy Management Configuration section, click the edit icon next to Premium Calculation Expression Set and Expected Revenue Expression Set and choose the respective expression sets.
3.  Save your work.
