Define whether tier structure is applicable for a coverage specification.

| To set up brokerage: | Insurance Brokerage User |

1.  In Setup, find and select **Brokerage**. Select **Policy Management**, and then select **Coverage Benefits**.
2.  In the Insurance Policy Coverage Spec Configuration, click **Add**.
3.  Select a coverage specification name.
4.  In the Tier Configuration field, enable or disable the tier applicability.
5.  Save your work.
