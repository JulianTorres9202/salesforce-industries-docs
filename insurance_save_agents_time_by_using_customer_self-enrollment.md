# Save Agents Time by Using Customer Self-Enrollment

Based on the contract and members group class, enable customers to view and select one or more eligible plans along with optional coverages, and then rate and enroll. Show the member level premium along with employee and employer contributions for a policy before enrollment. Customers can enroll right away or save the selections for enrolling later.

**Where**: This change applies to Lightning Experience in Professional, Enterprise, and Unlimited editions with the Insurance Industries managed package and the Insurance Industries Extension managed package.

**How**: Use these new and enhanced services for self-enrollment:

-   `InsCensusServiceStd:getMembers`
-   `InsCensusServiceStd:getMembersWithPlans`
-   `InsEnrollmentService:getRatedGroupProducts`
-   `InsGroupClassService:GetGroupClassesByContract`
-   `InsEnrollmentService:enrollFamily`
