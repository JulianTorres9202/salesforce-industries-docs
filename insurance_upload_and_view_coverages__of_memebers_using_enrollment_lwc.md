# Upload and View the Plan Coverage Details of Members for Enrollment

Upload group benefits coverage selections of members and view the coverage selections for each member in the user interface by using the Enrollment Census Lightning web component.

**Where:** This change applies to Lightning Experience in Professional, Enterprise, and Unlimited editions with the Insurance Industries managed package.

**How:** Include plan coverage details in the CSV file with enrollment data. After you upload the file, click the action menu for any member and select View Plan Coverage.
