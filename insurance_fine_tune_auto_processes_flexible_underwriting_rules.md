# Fine-Tune Automated Processes with Flexible Underwriting Rules

Apply underwriting rules to specific types of insured items or parties in a quote or insurance policy. Trigger unique messaging and actions based on whether each rule passes or fails.

**Where:** These changes apply to Lightning Experience in Professional, Enterprise, and Unlimited editions with the Insurance Industries managed package.

**Who:** Users need the Policy Administration add-on license to use this feature

**Why:** With rule scope defined, processes run more efficiently and accurately because each rule runs for exactly the right line items in a quote or insurance policy. For example, a rule that evaluates the monetary value of a vehicle runs just for vehicles, not for drivers.

**How:** Set the Rule Scope on the root product's Underwriting Rule. The product's Insured Items determine the available rule scopes.
