# Configure FlexCard-Based Claim 360 in Lightning Experience

In Lightning App Builder, drag FlexCard from the Components panel to the Lightning page canvas location where you want to position the component.

1.  In the Properties pane, search for a component in the FlexCard Name field, and then select the card.
2.  Save your changes.
3.  If your page is new, activate the page.
