# insQuoteCoverageList Lightning Web Component

This component displays all the coverages for a quote.

[](https://help.salesforce.com/s?language=en_US)

[](https://help.salesforce.com/s?language=en_US)This component works for desktop and is mobile responsive.

[](https://help.salesforce.com/s?language=en_US)

Here's what it looks like in the user interface.

Here are details of what exactly users can see in this LWC:

1.  An icon for the coverage
    
    Add images to the Files object on the product and that image appears as the icon
    
2.  Coverage name
    
3.  Premium price of the coverage
    
4.  Attribute name (one or more)
    
5.  Attribute value
    
    Hidden attributes don't show up here.
    

[](https://help.salesforce.com/s?language=en_US)

## What Users Can Do on this LWC

See all the coverages, their limits and deductibles for a quote.

[](https://help.salesforce.com/s?language=en_US)

## What this LWC Does

**insQuoteCoverageList** serves as a container for the different coverages.

[](https://help.salesforce.com/s?language=en_US)

## Services Called by this LWC

**insQuoteCoverageList** doesn't call any services.

[](https://help.salesforce.com/s?language=en_US)
