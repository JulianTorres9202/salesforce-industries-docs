# Insurance View Quote Permission Set

Give your agents access to view quotes with the Insurance View Quote permission set.

[](https://help.salesforce.com/s?language=en_US)

## Access Provided by the Insurance View Quote Permission Set

This permission set is delivered in the Insurance managed package and provides read-only access to the following objects:

[](https://help.salesforce.com/s?language=en_US)

-   Account
    
-   Attribute Assignment
    
-   Compiled Attribute Override
    
-   Contact
    
-   Contract
    
-   Group Census
    
-   Jurisdiction
    
-   Opportunity
    
-   Pricebook
    
-   Price List
    
-   Price List Entry
    
-   Producer Commission
    
-   Product
    
-   Product Child Item
    
-   Product Requirement
    
-   Quote
    
-   Quote Item Relationship
    
-   Quote Line Item
    
-   Quote Line Item Group Class
    
-   Quote Pricing Adjustment
    

Additionally, this permission set grants read-only access to certain fields on some of the objects listed above.
