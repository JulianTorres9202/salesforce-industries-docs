# Uninstall the Insurance FSC Unmanaged Package

Uninstall the Insurance FSC Unmanaged Package so that you are ready to install future versions of this package when they are released.

The highlighted area in this flow shows where you are in the process of setting up your Experience site:

1.  From Setup, enter Installed Packages in the Quick Find box, then select **Installed Packages**.
2.  On the Installed Packages page, locate the Insurance FSC Unmanaged package.
3.  Next to the unmanaged package, click **Uninstall**.
4.  Scroll down to the bottom of the page and choose to save or not save the package data for 48 hours.
5.  Select **Yes, I want to uninstall this package and permanently delete all associated components**.
6.  Click **Uninstall**.
