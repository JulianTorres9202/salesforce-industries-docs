# Verify Omnistudio Settings

Enable Metadata and disable Managed Package Runtime settings to verify the Omnistudio configuration.

-   Enable the Omnistudio Metadata setting. For more information, see ​[Enable Omnistudio Metadata API Support](https://help.salesforce.com/s/articleView?id=xcloud.os_enable_omnistudio_metadata_api_support.htm&language=en_US&type=5)
-   Disable the OmniStudio Managed Package Runtime setting. For more information, see ​[Disable Managed Package Runtime](https://help.salesforce.com/s/articleView?id=xcloud.os_enable_standard_omnistudio_runtime.htm&language=en_US&type=5)​
