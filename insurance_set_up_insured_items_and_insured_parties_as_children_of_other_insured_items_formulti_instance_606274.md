# Set Up Insured Items and Insured Parties as Children of Other Insured Items for Multi-Instance

You can create a parent-child relationship for insured items. This works a little bit differently than associating coverages with insured items.

1.  On the Insured Items tab, add the insured items you want to set up as parent and children.
2.  Click the icon on the gray bar of the child insured item. The cursor becomes a + sign with arrows.
3.  Drag the child insured item under, and a bit to the right of, the parent item.
    
    Note
    
    You can add a child insured item or party to only one parent insured item on a given root product spec.
