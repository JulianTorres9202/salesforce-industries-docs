# Open Claims

Use the Open Claims Data Processing Engine template to get a list of open claims.

The Open Claims Data Processing Engine template includes these key nodes:

| Node Name | Node Type | Description |
| --- | --- | --- |
| Claims | Data Source | The data source for claims. |
| Accounts | Data Source | The data source for accounts. |
| Status | Filter | Filters claims that are in Open status. |
| Open Claims | Writeback Object | The dataset containing open claims. |
| Claims and Accounts | Join | Joins claim details with account and policy details. |
