# Add the Create New Version Button to Calculation Matrix and Calculation Procedure Version Pages

The buttons that let you create new versions of calculation matrices and calculation procedures exist in Insurance, but they're not included in the page layouts by default.

[](https://help.salesforce.com/s?language=en_US)Your org might already have the Create New Version buttons included on Page Layouts. Check to see if it does. If it doesn't, you can add it.

1.  Add the **Create New Version** button to the Vlocity Calculation Matrix Version page.
    1.  From **Setup**, click the **Object Manager** tab.
    2.  Click **Vlocity Calculation Matrix Version**.
    3.  Click **Page Layouts**.
    4.  Click **Vlocity Pricing Matrix Version Layout**.
    5.  In the top panel, click **Mobile & Lightning Actions**.
    6.  In the main panel, under **Salesforce Mobile and Lightning Experience Actions**, click **Override the predefined actions**.
    7.  Drag **Create New Version** from the top panel down into the **Salesforce Mobile and Lightning Experience Actions** area.
    8.  Save your changes.
2.  Add the **Create New Version** button to the Vlocity Calculation Procedure Version page.
    1.  From **Setup**, click the **Object Manager** tab.
    2.  Click **Vlocity Calculation Procedure Version**.
    3.  Click **Page Layouts**.
    4.  Click **Pricing Procedure Version Layout**.
    5.  n the top panel, click **Mobile & Lightning Actions**.
    6.  In the main panel, under **Salesforce Mobile and Lightning Experience Actions**, click **Override the predefined actions**.
    7.  Drag **Create New Version** from the top panel down into the **Salesforce Mobile and Lightning Experience Actions**.
    8.  Save your changes.
