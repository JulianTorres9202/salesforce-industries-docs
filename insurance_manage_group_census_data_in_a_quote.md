# Manage Group Census Data in a Quote

Browse through details about group members, dependents, and plan selections. Add, edit, or delete participant data as needed.

1.  To take a quick look at census stats, click the information icon under Total Members. A Salesforce admin can customize the summary information that appears here.
2.  Scroll through detailed census data. Expand members to view their dependents and review details for each plan participant. An admin can customize the columns of data in the hierarchical tree view.
3.  Manage data as needed.
    
    -   Click **Add Member** to add a new member.
        
    -   Click the **Download Template** action to download a spreadsheet template to use for census data uploads.
        
    -   Click the action dropdown for an existing member to add a dependent or edit member data.
        
    -   Click  to delete one or more members from the quote census. Select at least one member record to enable the delete members icon.
        
    -   To quickly find specific members, search for them by member name. To learn more about the Search functionality, see [Search for Large Group Census Members](https://help.salesforce.com/s/articleView?id=ind.insurance_search_for_large_group_cecnsus_members.htm&language=en_US&type=5 "Quickly find census member details during large group enrollment and quoting by using the Search box in the Enrollment Census Lightning Web component and Quote Census Lightning web component respectively.").
