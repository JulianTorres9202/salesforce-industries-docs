# Contract Business Component Update

You can now set the time period for which the contract is valid in the Contract Creation OmniScript.

**Where:** You can find the Contract Creation OmniScript in the Vlocity OmniScript Designer under:

-   Type: InsGVB
    
-   SubType: CreateContract
    
-   OmniScript Name: Create Contract
    

**Why:** We added a new component for use in the Contract Creation OmniScript called `setContractTerm`. This component helps you set the time period for which the contract is valid.
