# Configure the Story Object for a History Timeline

You configure your History timeline by configuring settings on the story object. You can add a new set of data to the story object configuration, or edit an existing one.

1.  [](https://help.salesforce.com/s?language=en_US)From Setup, use Quick Find to find and select Custom Settings.
2.  [](https://help.salesforce.com/s?language=en_US)In the index across the top, click the letter **S**, then find **Story Object Configuration**, then click **Manage** next to it.
3.  [](https://help.salesforce.com/s?language=en_US)Click **New** or click **Edit** next to a story object setting.
    
    [](https://help.salesforce.com/s?language=en_US)If you add a new record type to the story object, associate it with an icon. This icon appears next to the `Title` field value on the History tab.
