# Manage Group Census Data

Browse and update information about group members, dependents, and plan selections. The fields you see when adding or editing a member or dependent are based on the default page layout of the Group Census Member object.

-   Add Member: Manually add an individual member to the census. Provide details such as name, first name, last name, and association with the primary member.
-   Add Dependent: From a member’s action menu, select Add Dependent. Provide details such as name, first name, last name, and association with the primary member. The dependent is automatically linked to the selected member.
-   Edit Member or Edit Dependent: From the action menu, select Edit Member or Edit Dependent. You can update personal details, association with the primary member, and other information. You can’t update Group Census or Group Class fields.
-   Delete Member: Select one or more group census member records Click the delete icon to remove from the census. Deleted records can’t be restored. Deleting primary members records deletes all its associated records.
-   Configure Plans (Enrollment Census Only): If the census type is Enrollment, open a member record to view and configure plan selections. You can assign or update plans and coverages for each member and dependent.
-   View Member Details:
    -   Grid: Scroll through census data. Expand a member to see dependents and their plan details. Admins can customize grid columns.
    -   Side Panel: Click a member to view their Group Census Member details in the side panel. Admins can customize displayed fields.
    -   Search Members: Find members quickly by searching for their first or last name.
