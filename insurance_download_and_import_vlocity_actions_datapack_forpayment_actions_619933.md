# Download and Import Vlocity Actions DataPack for Payment Actions

Get a head-start configuring custom payment actions that invoke your own front- and back-end processes when users click specific payment actions. Download and import a DataPack with Vlocity Actions to customize or use as templates.

1.  [](https://help.salesforce.com/s?language=en_US)From Setup, enter Static Resources in the Quick Find box, then select **Static Resources**.
2.  [](https://help.salesforce.com/s?language=en_US)Find and click **DP\_CUSTOM\_ClaimVlocityAction**.
3.  [](https://help.salesforce.com/s?language=en_US)Right-click **View File** and select **Save Link As**.
4.  [](https://help.salesforce.com/s?language=en_US)Add .json to the end of the file name suggested, so the full name is vlocity\_ins\_\_DP\_CUSTOM\_ClaimVlocityAction.json. Browse to a folder that's easy to find on your local computer, then click **Save**.
5.  [](https://help.salesforce.com/s?language=en_US)From the App Launcher, find and select **OmniStudio DataPacks**.
6.  On the **Installed** tab, choose to import from a file.
7.  Next to the File text box, click **Browse** to navigate to a local copy of the resource that you want to upload. Then click **Next**.
8.  Make sure all the items are selected, then click **Next**.
9.  Click **Next**, then click **Done**.
10.  Click **Activate Now**, then click **Done**.
11.  [](https://help.salesforce.com/s?language=en_US)From the App Launcher, find and select **Vlocity Actions** and confirm that the list includes **PaymentDetailRequestExGratiaAction** and **PaymentDetailRequestAuthorityAction**.
