# Customize the FlexCard-Based Claim 360 Component

Customize the OmniStudio FlexCards, Integration Procedures, and Omnistudio Data Mappers to suit your business needs.

To start using the FlexCard-based Claim 360 component, customize the OmniStudio FlexCards, Integration Procedures, and Data Mappers, and then configure the component in Lightning Experience.

-   **[Customize FlexCard-Based Claim 360 Using FlexCards](https://help.salesforce.com/s/articleView?id=ind.insurance_customize_flexcardbased_claim_360_using_flexcards.htm&language=en_US&type=5)**  
    To show claim information with clickable actions that change based on the information entered, customize the FlexCard components that you want to add to the record pages.
-   **[Customize FlexCard-Based Claim 360 by Using Integration Procedures](https://help.salesforce.com/s/articleView?id=ind.insurance_customize_flexcardbased_claim_360_by_using_integration_procedures.htm&language=en_US&type=5)**  
    Customize integration procedures in OmniStudio.
-   **[Customize FlexCard-Based Claim 360 Using Data Mappers](https://help.salesforce.com/s/articleView?id=ind.insurance_customize_flexcardbased_claim_360_using_datamappers.htm&language=en_US&type=5)**  
    If you want to include policy and claim details from a third-party system, get your developer to create an Apex class that makes the data available to FlexCard-based Claim 360. Then, customize the Omnistudio Data Mappers to reference the Apex class.
