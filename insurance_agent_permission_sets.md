# Agent Permission Sets

Give your agents access to just what they need to complete their jobs with Agent Permission Sets.

**Where:** Pre-made permission sets designed for your agents are delivered in the Summer '21 Vlocity Insurance managed package and in the Summer '21 Insurance Industries Extension managed package.

**Why:** Agents really only need access to a few objects, and to only certain fields on those objects. We've created permission sets that you can assign to agents to give them access only to the information they need to help their customers succeed-- and nothing else.

**How:** We've created eight Agent permission sets for you. Each permission set grants a different level of access to a different set of objects and fields. Assign the permission set (or sets) to your agents to provide them with the correct level of access.
