# Add Picklist Values for Insurance Industries Extension Releases Spring '21 Through Winter '25

Add values to picklist fields on several objects used with the Insurance Industries Extension.

1.  From **Setup**, click the **Object Manager** tab.
2.  Click an object from the list below.
3.  Click **Fields & Relationships**.
4.  Click the field listed for that object.
5.  In the **Values** section, click **New**.
6.  On the Add Picklist Values page, enter the picklist value in the text field.
    
    If you're adding multiple values, enter each value on a new line.
    
7.  Click **Save**.
    
    Use these steps to add all of the picklist values listed here.
    
    | Object | Field | Entry | Release Added |
    | --- | --- | --- | --- |
    | Insurance Policy | Status | Reversed | Summer '22 |
    | Product Requirement | Applicable Object | Insurance Policy | Summer '22 |
    | Vlocity State Model | Object API Name | Claim | Spring '21 |
    | Insurance Policy | Summer '22 |
