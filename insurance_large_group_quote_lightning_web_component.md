# Large Group Quote Lightning Web Component

The Large Group Quote LWC is specially designed to help you build complex quotes. It features a side panel for editing line items. Buttons for common actions display prominently so your users can issue policies, create quote versions, compare quote version histories, and add additional plans with ease.

This component supports multi-root quotes. A multi-root quote is a quote that has multiple root products. Users can view root products or plans and associated coverages on the same page.

Note The Large Group Quote LWC is designed to support synchronous and asynchronous rating for large group quoting. For small and medium group quoting, use the insQuote LWC. Read [insQuote Lightning Web Component](https://help.salesforce.com/s/articleView?id=ind.insurance_insquote_lwc.htm&language=en_US&type=5 "Use this component to display a quote. Calling other Lighting web components, insQuote, called Vlocity Insurance Quote in the Lightning App Builder, displays root products, coverages, insured items, insured parties, and attributes.") to learn how.

-   [Large Group Rating](https://help.salesforce.com/s/articleView?id=ind.insurance_large_group_rating.htm&language=en_US&type=5#insurance_large_group_rating "Rating for large groups is based on the rating procedure defined on the plans included in the quote. Rating procedures are created in the Business Rules Engine as Expression Sets.")
