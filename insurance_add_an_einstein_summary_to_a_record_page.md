# Add an Einstein Summary to a Record Page

Enable your customer service representatives (CSRs) or captive agents to quickly generate a summary of the information stored in Account or Claim records by using Einstein Summary. To use Einstein Summary, add the Einstein Summary component to an Account or Claim record page.

1.  From Setup, in the Quick Find box, enter Lightning App Builder, and then select **Lightning App Builder**.
2.  Select the record page that you want to modify and click **Edit**.
3.  Drag the Einstein Summary component to the record page.
4.  Under Prompt Template Type, click **Select**.
5.  From the Available column, drag the **Record Summary** prompt template type to the Selected column.
6.  Click **OK**.
7.  On Prompt Template Name, click **Select**.
8.  From the Available column, drag the prompt template to the Selected column.
9.  Click **OK**.
10.  Save your changes, and then activate the page.
