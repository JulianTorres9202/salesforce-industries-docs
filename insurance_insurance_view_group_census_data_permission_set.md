# Insurance View Group Census Data Permission Set

Give your agents access to just what they need to view group census data.

[](https://help.salesforce.com/s?language=en_US)

## Access Provided by the Insurance View Group Census Data Permission Set

This permission set is delivered in the Insurance Industries Extension managed package and provides read-only access to the following objects:

[](https://help.salesforce.com/s?language=en_US)

-   Account
    
-   Attribute
    
-   Attribute Assignment
    
-   Attribute Category
    
-   Compiled Attribute Override
    
-   Contact
    
-   Contract
    
-   Contract Group Plan
    
-   Group Census
    
-   Group Census Member
    
-   Group Class
    
-   Group Class Contribution
    
-   Insurance Contract
    
-   Pricebook
    
-   Price List
    
-   Price List Entry
    
-   Product
    
-   Product Child Item
    

Additionally, this permission set grants read-only access to certain fields on some of the objects listed above.
