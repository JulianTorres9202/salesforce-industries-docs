# Replace the Data Source on the FlexCard

If you need to, you can replace the data source on a FlexCard with another data source:

1.  Deactivate the FlexCard.
2.  On the FlexCard for the component you're managing, on the right pane click **Setup**.
3.  Scroll down to **Data Source**. In the **Data Source Type** menu, choose a type.
4.  Depending on the **Data Source Type** you chose, fill out the name of the data source and additional inputs and options as needed.
5.  (Optional) In the **Test Parameters** section, enter **Keys** and **Values** that will let you test this FlexCard.
    
    Tip
    
    We recommend that you set up **Test Parameters** if your data source takes parameters. Elements like **Field**, **Action**, and **Condition** all reference data from the data source. When you can see real data pulled into the FlexCard, it's much easier to compose and edit the FlexCard in the Designer.
    
    Test parameters don't affect the runtime or preview of a component.
    
6.  Click **Save & Fetch**.
7.  Activate the FlexCard.
