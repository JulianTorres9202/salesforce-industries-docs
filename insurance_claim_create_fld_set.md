# Create Field Sets

Salesforce has a drag-and-drop WYSIWYG tool for creating field sets.

1.  From the management settings for the Claim Coverage Payment Detail object, go to Field Sets, and then click **New**.
2.  Enter a Field Set Label.
3.  Provide a brief description of which pages use the field set, and for what purpose.
4.  Save your changes.
5.  To add fields to the field set, drag the fields from the object palette and drop them In the Field Set container.
6.  Save your changes.
